¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets) es el lenguaje que se utiliza para describir la presentación de documentos HTML. Mientras HTML se encarga de la estructura y el contenido, CSS se encarga del diseño, colores, tipografías, espaciado y disposición visual de los elementos.

¿Por qué separar HTML de CSS?

  • Separación de responsabilidades: HTML para contenido, CSS para presentación
  • Reutilización: Un mismo CSS puede aplicarse a múltiples páginas HTML
  • Mantenimiento: Es más fácil actualizar el diseño sin tocar el contenido
  • Carga eficiente: Los archivos CSS se descargan una vez y se almacenan en caché
  • Flexibilidad: Diferentes estilos para diferentes dispositivos o contextos

Formas de aplicar CSS

Existen tres maneras principales de incluir CSS en un documento HTML:

1. CSS Inline (en línea)

Se aplica directamente en el elemento HTML usando el atributo style:

<p style="color: blue; font-size: 18px;">
  Este párrafo tiene estilos en línea
</p>

<h1 style="background-color: yellow; text-align: center;">
  Título con fondo amarillo
</h1>

Ventajas: Rápido para pruebas puntuales
Desventajas: Difícil de mantener, mezcla contenido con presentación

2. CSS Interno (en etiqueta <style>)

Se incluye dentro del <head> del documento HTML:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi página con CSS interno</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      background-color: #f0f0f0;
    }
    
    h1 {
      color: #333;
      text-align: center;
    }
    
    p {
      color: #666;
      line-height: 1.6;
    }
  </style>
</head>
<body>
  <h1>Título Principal</h1>
  <p>Este párrafo usa CSS interno.</p>
</body>
</html>

Ventajas: Todos los estilos en un lugar, fácil para páginas pequeñas
Desventajas: No se puede reutilizar en otras páginas

3. CSS Externo (archivo separado) - ¡RECOMENDADO!

Los estilos se guardan en un archivo separado con extensión .css y se enlazan al HTML:

Archivo HTML (index.html):

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi página con CSS externo</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h1>Título Principal</h1>
  <p>Este párrafo usa CSS externo.</p>
</body>
</html>

Archivo CSS (styles.css):

/* Este es un comentario en CSS */
body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
  margin: 0;
  padding: 20px;
}

h1 {
  color: #333;
  text-align: center;
  margin-bottom: 20px;
}

p {
  color: #666;
  line-height: 1.6;
  font-size: 16px;
}

Ventajas: Reutilizable, mantenible, cacheable, mejor organización
Desventajas: Requiere un archivo adicional

Anatomía de una regla CSS

Una regla CSS tiene la siguiente estructura:

selector {
  propiedad: valor;
  otra-propiedad: otro-valor;
}

/* Ejemplo real */
h1 {
  color: blue;
  font-size: 24px;
  text-align: center;
}
  • Selector: Indica qué elementos HTML se van a estilizar
  • Propiedad: Qué aspecto se va a modificar (color, tamaño, etc.)
  • Valor: Cómo se va a modificar esa propiedad
  • Declaración: Cada línea propiedad: valor;
  • Bloque de declaraciones: Todo lo que está entre las llaves { }

Comentarios en CSS

Los comentarios en CSS se escriben entre /* y */:

/* Este es un comentario de una línea */

/*
  Este es un comentario
  de múltiples líneas
  muy útil para documentar
*/

body {
  font-family: Arial, sans-serif; /* Fuente principal */
  /* background-color: red; Esta línea está comentada */
}

Buenas prácticas desde el inicio

  • ✅ Usa CSS externo siempre que sea posible
  • ✅ Organiza tu código con comentarios y espacios
  • ✅ Usa nombres descriptivos para clases e IDs
  • ✅ Mantén consistencia en el estilo de escritura
  • ✅ Una declaración por línea para mejor legibilidad
  • ❌ Evita CSS inline para proyectos reales
  • ❌ No uses estilos duplicados innecesariamente

Selectores básicos en CSS

Los selectores son patrones que nos permiten seleccionar los elementos HTML a los que queremos aplicar estilos. Existen diferentes tipos de selectores, cada uno con su propósito específico.

1. Selector de elemento (o tipo)

Selecciona todos los elementos de un tipo específico:

CSS:

/* Selecciona TODOS los párrafos */
p {
  color: blue;
  font-size: 16px;
}

/* Selecciona TODOS los títulos h1 */
h1 {
  color: red;
  text-align: center;
}

/* Selecciona TODOS los enlaces */
a {
  text-decoration: none;
  color: green;
}

HTML correspondiente:

<h1>Este título será rojo y centrado</h1>
<p>Este párrafo será azul</p>
<p>Este párrafo también será azul</p>
<a href="#">Este enlace será verde sin subrayado</a>

2. Selector de clase (.clase)

Selecciona elementos que tienen un atributo class específico. Se escribe con un punto (.) seguido del nombre de la clase:

CSS:

/* Selecciona elementos con class="destacado" */
.destacado {
  background-color: yellow;
  font-weight: bold;
}

/* Selecciona elementos con class="boton" */
.boton {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
}

/* Selecciona elementos con class="texto-pequeño" */
.texto-pequeño {
  font-size: 12px;
  color: gray;
}

HTML correspondiente:

<p>Este párrafo es normal</p>
<p class="destacado">Este párrafo está destacado</p>
<button class="boton">Botón estilizado</button>
<span class="texto-pequeño">Texto en tamaño pequeño</span>

<!-- Un elemento puede tener múltiples clases -->
<p class="destacado texto-pequeño">Párrafo destacado y pequeño</p>

3. Selector de ID (#id)

Selecciona el elemento que tiene un atributo id específico. Se escribe con una almohadilla (#) seguida del nombre del ID:

CSS:

/* Selecciona el elemento con id="encabezado" */
#encabezado {
  background-color: navy;
  color: white;
  padding: 20px;
  text-align: center;
}

/* Selecciona el elemento con id="menu-principal" */
#menu-principal {
  background-color: #f8f9fa;
  border: 1px solid #ddd;
  padding: 15px;
}

/* Selecciona el elemento con id="pie-pagina" */
#pie-pagina {
  background-color: #333;
  color: white;
  text-align: center;
  padding: 10px;
}

HTML correspondiente:

<header id="encabezado">
  <h1>Mi Sitio Web</h1>
</header>

<nav id="menu-principal">
  <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Acerca</a></li>
  </ul>
</nav>

<footer id="pie-pagina">
  <p>© 2025 Mi Sitio Web</p>
</footer>

4. Selector universal (*)

Selecciona TODOS los elementos de la página:

/* Aplicar a TODOS los elementos */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

5. Selectores de agrupación

Puedes aplicar los mismos estilos a múltiples selectores separándolos con comas:

/* Aplicar los mismos estilos a h1, h2 y h3 */
h1, h2, h3 {
  color: #333;
  font-family: 'Georgia', serif;
  margin-bottom: 15px;
}

/* Aplicar estilos a múltiples clases */
.boton, .enlace-boton, .btn-primary {
  background-color: blue;
  color: white;
  padding: 10px 15px;
  text-decoration: none;
  border-radius: 5px;
}

Especificidad de selectores

Cuando varios selectores apuntan al mismo elemento, CSS usa un sistema de especificidad para determinar qué estilos aplicar:

Orden de especificidad (de menor a mayor):

/* 1. Selector de elemento (especificidad: 1) */
p {
  color: black;
}

/* 2. Selector de clase (especificidad: 10) */
.texto-azul {
  color: blue;
}

/* 3. Selector de ID (especificidad: 100) */
#texto-especial {
  color: red;
}

/* 4. CSS Inline (especificidad: 1000) */
/* <p style="color: green;"> */

Ejemplo práctico de especificidad:

<!-- HTML -->
<p id="texto-especial" class="texto-azul">
  ¿De qué color será este texto?
</p>

<!-- CSS -->
p { color: black; }           /* Especificidad: 1 */
.texto-azul { color: blue; }  /* Especificidad: 10 */
#texto-especial { color: red; } /* Especificidad: 100 - GANA */

/* Resultado: el texto será ROJO */

Buenas prácticas para selectores

  • ✅ Usa clases para estilos reutilizables: .boton, .destacado
  • ✅ Usa IDs para elementos únicos: #encabezado, #menu-principal
  • ✅ Nombres descriptivos: .boton-primario en lugar de .azul
  • ✅ Selectores de elemento para estilos globales: body, h1, p
  • ❌ Evita IDs para estilos reutilizables
  • ❌ No uses selectores demasiado específicos sin necesidad

Propiedades CSS fundamentales

CSS ofrece cientos de propiedades para controlar la apariencia de los elementos. Aquí aprenderás las propiedades más importantes y utilizadas en el desarrollo web.

Propiedades de color

Controlan el color del texto y del fondo de los elementos:

color (color del texto)

/* Diferentes formas de especificar colores */

/* Nombres de colores predefinidos */
p { color: red; }
h1 { color: blue; }
span { color: green; }

/* Colores hexadecimales */
.titulo { color: #ff0000; }    /* Rojo */
.subtitulo { color: #0066cc; } /* Azul */
.texto { color: #333333; }     /* Gris oscuro */

/* Colores RGB */
.header { color: rgb(255, 0, 0); }      /* Rojo */
.nav { color: rgb(0, 102, 204); }       /* Azul */

/* Colores RGBA (con transparencia) */
.overlay { color: rgba(0, 0, 0, 0.7); } /* Negro con 70% opacidad */

/* Colores HSL */
.banner { color: hsl(120, 100%, 50%); } /* Verde puro */

background-color (color de fondo)

/* Color de fondo sólido */
body { background-color: #f8f9fa; }
.destacado { background-color: yellow; }
.boton { background-color: #007bff; }

/* Fondo transparente */
.modal { background-color: rgba(0, 0, 0, 0.5); }

/* Quitar el fondo */
.sin-fondo { background-color: transparent; }

Propiedades de fuente y texto

font-family (tipo de fuente)

/* Fuente única */
body { font-family: Arial; }

/* Lista de fuentes (fallback) */
body { font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; }

/* Familias genéricas */
.serif { font-family: serif; }          /* Times New Roman, Georgia */
.sans-serif { font-family: sans-serif; } /* Arial, Helvetica */
.monospace { font-family: monospace; }   /* Courier New, Monaco */

/* Fuentes de Google Fonts */
.titulo { font-family: 'Roboto', sans-serif; }
.elegante { font-family: 'Playfair Display', serif; }

font-size (tamaño de fuente)

/* Unidades absolutas */
.pequeno { font-size: 12px; }
.normal { font-size: 16px; }
.grande { font-size: 24px; }

/* Unidades relativas */
.relativo { font-size: 1.2em; }    /* 1.2 veces el tamaño del padre */
.responsive { font-size: 1.5rem; }  /* 1.5 veces el tamaño raíz */

/* Porcentajes */
.porcentaje { font-size: 120%; }

/* Palabras clave */
.muy-pequeno { font-size: xx-small; }
.pequeno { font-size: small; }
.mediano { font-size: medium; }
.grande { font-size: large; }
.muy-grande { font-size: xx-large; }

font-weight (grosor de fuente)

/* Palabras clave */
.delgado { font-weight: normal; }   /* 400 */
.negrita { font-weight: bold; }     /* 700 */
.mas-delgado { font-weight: lighter; }
.mas-grueso { font-weight: bolder; }

/* Valores numéricos */
.muy-delgado { font-weight: 100; }
.delgado { font-weight: 300; }
.normal { font-weight: 400; }
.medio { font-weight: 500; }
.semi-negrita { font-weight: 600; }
.negrita { font-weight: 700; }
.extra-negrita { font-weight: 800; }
.ultra-negrita { font-weight: 900; }

text-align (alineación del texto)

.izquierda { text-align: left; }      /* Por defecto */
.centrado { text-align: center; }
.derecha { text-align: right; }
.justificado { text-align: justify; }  /* Alinea a ambos lados */

text-decoration (decoración del texto)

/* Quitar subrayado de enlaces */
a { text-decoration: none; }

/* Agregar decoraciones */
.subrayado { text-decoration: underline; }
.tachado { text-decoration: line-through; }
.sobrayado { text-decoration: overline; }

/* Combinar decoraciones */
.decorado { text-decoration: underline overline; }

Modelo de caja básico

margin (margen exterior)

/* Margen uniforme */
.espaciado { margin: 20px; }

/* Margen específico por lado */
.margen-top { margin-top: 10px; }
.margen-right { margin-right: 15px; }
.margen-bottom { margin-bottom: 10px; }
.margen-left { margin-left: 15px; }

/* Margen con 2 valores (vertical horizontal) */
.margen-vh { margin: 10px 20px; }

/* Margen con 4 valores (top right bottom left) */
.margen-completo { margin: 10px 15px 20px 5px; }

/* Centrar elemento horizontalmente */
.centrado { margin: 0 auto; }

padding (espaciado interior)

/* Padding uniforme */
.relleno { padding: 15px; }

/* Padding específico por lado */
.relleno-top { padding-top: 10px; }
.relleno-right { padding-right: 20px; }
.relleno-bottom { padding-bottom: 10px; }
.relleno-left { padding-left: 20px; }

/* Padding con múltiples valores */
.relleno-vh { padding: 10px 20px; }           /* vertical horizontal */
.relleno-completo { padding: 5px 10px 15px 20px; } /* top right bottom left */

border (bordes)

/* Borde simple */
.con-borde { border: 1px solid black; }

/* Borde detallado */
.borde-completo {
  border-width: 2px;
  border-style: solid;
  border-color: #007bff;
}

/* Bordes específicos por lado */
.borde-superior { border-top: 2px solid red; }
.borde-derecho { border-right: 1px dashed blue; }
.borde-inferior { border-bottom: 3px dotted green; }
.borde-izquierdo { border-left: 2px solid purple; }

/* Estilos de borde */
.solido { border: 2px solid black; }
.punteado { border: 2px dotted red; }
.rayado { border: 2px dashed blue; }
.doble { border: 4px double green; }

/* Bordes redondeados */
.redondeado { border-radius: 5px; }
.muy-redondeado { border-radius: 15px; }
.circular { border-radius: 50%; }

Propiedades de fondo avanzadas

background (propiedades de fondo)

/* Imagen de fondo */
.con-imagen {
  background-image: url('imagen.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

/* Fondo con gradiente */
.gradiente {
  background: linear-gradient(to right, #ff7e5f, #feb47b);
}

/* Múltiples fondos */
.multiple {
  background: 
    url('patron.png') repeat,
    linear-gradient(to bottom, #fff, #f0f0f0);
}

Ejemplo completo combinando propiedades

HTML:

<div class="tarjeta">

.tarjeta {
  /* Modelo de caja */

  margin: 20px auto;
  padding: 20px;
  border: 1px solid #ddd;

  /* Fondo */
  background-color: #ffffff;

  font-family: 'Arial', sans-serif;

.titulo-tarjeta {
  /* Color y fuente */
  color: #333;
  font-size: 24px;
  font-weight: bold;
  
  /* Alineación y espaciado */
  text-align: center;
  margin-bottom: 15px;

  /* Texto */

  font-size: 16px;
  line-height: 1.5;
  text-align: justify;

  margin-bottom: 20px;
}

  /* Apariencia de botón */

  padding: 10px 20px;
  border-radius: 5px;

  text-decoration: none;
  font-weight: bold;

Unidades de medida en CSS

Unidades absolutas:

/* Píxeles (más común) */
.pixeles { font-size: 16px; margin: 10px; }

/* Otras unidades absolutas (menos comunes) */
.puntos { font-size: 12pt; }      /* Puntos (impresión) */
.pulgadas { width: 2in; }         /* Pulgadas */
.centimetros { width: 5cm; }      /* Centímetros */

Unidades relativas:

/* em - relativo al tamaño de fuente del padre */
.em-unit { font-size: 1.2em; padding: 0.5em; }

/* rem - relativo al tamaño de fuente raíz */
.rem-unit { font-size: 1.5rem; margin: 2rem; }

/* Porcentajes - relativo al elemento padre */
.porcentaje { width: 100%; height: 50%; }

/* Unidades de viewport */
.viewport-width { width: 100vw; }    /* 100% del ancho de pantalla */
.viewport-height { height: 100vh; }  /* 100% del alto de pantalla */

Jerarquía y herencia

Algunas propiedades CSS se heredan automáticamente de los elementos padre:

/* Propiedades que se heredan */
body {
  font-family: Arial, sans-serif;  /* Se hereda a todos los hijos */
  color: #333;                     /* Se hereda a todos los hijos */
  line-height: 1.6;              /* Se hereda a todos los hijos */
}

/* Los elementos hijos heredarán estas propiedades automáticamente */
/* Pero pueden sobrescribirse específicamente */
h1 {
  color: #007bff;  /* Sobrescribe el color heredado */
}

Ejercicio Práctico

Crear tu primer archivo CSS

Intermedio

Objetivo: Crea un archivo styles.css y enlázalo con tu HTML. Aplica color de fondo, tipo de fuente y formato de texto.

Pasos a seguir:

  1. Crea un archivo llamado styles.css
  2. Enlaza el archivo CSS con tu HTML
  3. Aplica estilos al body (fondo y fuente)
  4. Estiliza encabezados y párrafos
  5. Usa clases e IDs para estilos específicos

Ejemplo HTML:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
    <title>Mi página con CSS</title>
</head>
<body>
    <h1 id="titulo-principal">Mi Primera Página con CSS</h1>
    <p class="descripcion">Este párrafo tiene estilos CSS.</p>
</body>
</html>

Ejemplo CSS (styles.css):

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    margin: 0;
    padding: 20px;
}

#titulo-principal {
    color: #333;
    text-align: center;
    border-bottom: 2px solid #007bff;
}

.descripcion {
    color: #666;
    font-size: 16px;
    line-height: 1.5;
    background-color: white;
    padding: 15px;
    border-radius: 5px;
}
