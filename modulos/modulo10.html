Crear un grid (display: grid)

CSS Grid Layout es un sistema de layout bidimensional que permite crear diseños complejos organizando elementos en filas y columnas simultáneamente.

¿Por qué usar CSS Grid?

  • ✅ Layout bidimensional (filas Y columnas)
  • ✅ Control preciso de posicionamiento
  • ✅ Layouts complejos con pocas líneas de código
  • ✅ Superposición natural de elementos
  • ✅ Responsive automático con funciones avanzadas
  • ✅ Separación contenido/diseño mejorada

Activando CSS Grid

Para convertir un elemento en contenedor grid:

/* Contenedor grid de bloque */
.grid-container {
  display: grid;
}

/* Contenedor grid inline */
.inline-grid-container {
  display: inline-grid;
}

Anatomía de CSS Grid

Grid trabaja con varios conceptos clave:

1. Grid Container (Contenedor Grid)

El elemento padre que se convierte en contenedor grid:

.grid-container {
  display: grid;
  /* Se convierte en contenedor grid */
}

2. Grid Items (Elementos Grid)

Los hijos directos del contenedor grid:

<div class="grid-container">
  <div class="grid-item">Item 1</div>  <!-- Grid item -->
  <div class="grid-item">Item 2</div>  <!-- Grid item -->
  <div class="grid-item">Item 3</div>  <!-- Grid item -->
</div>

3. Grid Lines (Líneas de Grid)

Las líneas que forman la estructura del grid:

/* Diagrama conceptual de líneas: */

     1    2    3    4
   ┌────┬────┬────┐  1
   │    │    │    │
   ├────┼────┼────┤  2
   │    │    │    │
   ├────┼────┼────┤  3
   │    │    │    │
   └────┴────┴────┘  4

/* Grid de 3x3 tiene:
   - 4 líneas verticales (1-4)
   - 4 líneas horizontales (1-4) */

4. Grid Tracks (Pistas de Grid)

El espacio entre dos líneas adyacentes (filas o columnas):

/* 3 columnas = 3 column tracks */
.grid {
  display: grid;
  grid-template-columns: 100px 200px 100px;
  /* Track 1: 100px, Track 2: 200px, Track 3: 100px */
}

5. Grid Cells (Celdas de Grid)

La intersección entre una fila y una columna:

/* Cada intersección es una celda */
/* Grid 3x3 = 9 celdas total */

┌─────┬─────┬─────┐
│ 1,1 │ 1,2 │ 1,3 │
├─────┼─────┼─────┤
│ 2,1 │ 2,2 │ 2,3 │
├─────┼─────┼─────┤
│ 3,1 │ 3,2 │ 3,3 │
└─────┴─────┴─────┘

6. Grid Areas (Áreas de Grid)

Grupos rectangulares de celdas:

/* Un área puede ocupar múltiples celdas */
.grid-item {
  grid-area: 1 / 1 / 3 / 3;  /* Fila 1-3, Columna 1-3 */
}

/* Resultado: área de 2x2 celdas */
┌─────────┬─────┐
│  ÁREA   │     │
│         │     │
├─────────┼─────┤
│         │     │
└─────────┴─────┘

Primer ejemplo básico

HTML:

<div class="basic-grid">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>

CSS:

.basic-grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;  /* 3 columnas iguales */
  grid-template-rows: 100px 100px;     /* 2 filas de 100px */
  gap: 10px;                           /* Espacio entre elementos */
  background-color: #f8f9fa;
  padding: 20px;
}

.item {
  background-color: #007bff;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 1.2rem;
  border-radius: 5px;
}

/* Resultado: Grid 3x2 con 6 elementos distribuidos automáticamente */

Unidades en CSS Grid

fr (fraction unit) - La unidad estrella

/* 1fr = una fracción del espacio disponible */
.grid-fractions {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;  /* 1:2:1 proporción */
  /* Total: 4fr
     Col 1: 1/4 = 25%
     Col 2: 2/4 = 50%  
     Col 3: 1/4 = 25% */
}

/* Combinando fr con otras unidades */
.grid-mixed {
  display: grid;
  grid-template-columns: 200px 1fr 100px;
  /* Col 1: 200px fijo
     Col 2: resto del espacio disponible
     Col 3: 100px fijo */
}

Otras unidades útiles

/* Unidades fijas */
.grid-fixed {
  grid-template-columns: 100px 200px 150px;  /* Píxeles */
  grid-template-columns: 20% 50% 30%;        /* Porcentajes */
  grid-template-columns: 10rem 5rem 8rem;    /* REM */
}

/* Auto - tamaño basado en contenido */
.grid-auto {
  grid-template-columns: auto 1fr auto;
  /* Col 1: tamaño del contenido
     Col 2: espacio restante
     Col 3: tamaño del contenido */
}

/* min-content y max-content */
.grid-content {
  grid-template-columns: min-content 1fr max-content;
  /* Col 1: mínimo espacio necesario
     Col 2: espacio restante
     Col 3: máximo espacio que querría */
}

Comportamiento por defecto

Al activar display: grid sin más propiedades:

.default-grid {
  display: grid;
  /* Sin más propiedades definidas */
}

/* Comportamiento automático:
   - Una sola columna (todos los elementos apilados)
   - Filas automáticas según el contenido
   - Sin espacios entre elementos (gap: 0)
   - Altura de fila = altura del contenido */

/* Ejemplo visual:
   [Item 1]
   [Item 2]
   [Item 3]
   [Item 4] */

Diferencias inmediatas con Flexbox

Comparación rápida:

/* FLEXBOX - Unidimensional */
.flex-container {
  display: flex;
  /* Elementos se alinean en UNA dirección */
  /* [Item1] [Item2] [Item3] */
}

/* GRID - Bidimensional */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  /* Elementos se organizan en filas Y columnas */
  /* [Item1] [Item2]
     [Item3] [Item4] */
}

/* Flexbox: óptimo para distribución lineal
   Grid: óptimo para layouts 2D complejos */

Ejemplo práctico: Layout de página básico

HTML estructura:

<div class="page-grid">
  <header class="header">Header</header>
  <nav class="sidebar">Sidebar</nav>
  <main class="content">Contenido principal</main>
  <footer class="footer">Footer</footer>
</div>

CSS con Grid:

.page-grid {
  display: grid;
  grid-template-columns: 200px 1fr;  /* Sidebar fijo, contenido flexible */
  grid-template-rows: auto 1fr auto; /* Header/footer auto, contenido flexible */
  min-height: 100vh;
  gap: 1rem;
}

/* Posicionamiento específico */
.header {
  grid-column: 1 / -1;  /* Ocupa todas las columnas */
  background-color: #343a40;
  color: white;
  padding: 1rem;
}

.sidebar {
  background-color: #f8f9fa;
  padding: 1rem;
}

.content {
  background-color: white;
  padding: 2rem;
}

.footer {
  grid-column: 1 / -1;  /* Ocupa todas las columnas */
  background-color: #6c757d;
  color: white;
  text-align: center;
  padding: 1rem;
}

/* Resultado: layout clásico con pocas líneas */

Ventajas inmediatas de Grid

✅ Comparado con Flexbox:

  • Control bidimensional vs unidimensional
  • Posicionamiento específico de elementos
  • Superposición natural
  • Layouts complejos sin nesting

✅ Comparado con floats/positioning:

  • No necesita clearfix
  • Responsive automático
  • Código más limpio y mantenible
  • No afecta al flujo normal del documento

Casos de uso ideales para Grid

/* ✅ IDEAL para Grid: */
- Layouts de página completos
- Galerías de imágenes
- Dashboard con widgets
- Formularios complejos
- Diseños de revista/periódico
- Cards con diferentes tamaños
- Superposición de elementos

/* ❌ Mejor usar Flexbox para: */
- Navegación horizontal simple
- Distribución de botones
- Centrado simple
- Elementos en línea
- Componentes lineales pequeños

Propiedades fundamentales de Grid

Las propiedades de Grid se dividen entre las del contenedor (grid container) y las de los elementos (grid items).

grid-template-columns: definiendo columnas

Especifica el tamaño y número de columnas en el grid:

/* Columnas de tamaño fijo */
.grid-fixed-columns {
  display: grid;
  grid-template-columns: 100px 200px 100px;  /* 3 columnas específicas */
}

/* Columnas flexibles con fr */
.grid-flex-columns {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;  /* Proporción 1:2:1 */
}

/* Combinando diferentes unidades */
.grid-mixed-columns {
  display: grid;
  grid-template-columns: 200px 1fr 150px;  /* Fijo + Flexible + Fijo */
}

/* Repetir patrones con repeat() */
.grid-repeat-columns {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* = 1fr 1fr 1fr */
  grid-template-columns: repeat(4, 100px); /* = 100px 100px 100px 100px */
}

/* Patrones complejos con repeat() */
.grid-complex-repeat {
  display: grid;
  grid-template-columns: repeat(2, 100px 200px);  /* = 100px 200px 100px 200px */
}

grid-template-rows: definiendo filas

Funciona igual que grid-template-columns pero para filas:

/* Filas específicas */
.grid-rows {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 200px 150px;  /* 3 filas específicas */
}

/* Filas automáticas vs específicas */
.grid-auto-rows {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px;  /* Solo primera fila definida */
  grid-auto-rows: 80px;       /* Filas adicionales serán de 80px */
}

/* Filas flexibles */
.grid-flexible-rows {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  grid-template-rows: auto 1fr auto;  /* Header, contenido, footer */
  height: 100vh;
}

gap: espaciado entre elementos

Controla el espacio entre filas y columnas:

/* Gap uniforme */
.grid-gap {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;  /* 20px entre filas y columnas */
}

/* Gap diferente para filas y columnas */
.grid-gap-different {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px 20px;  /* row-gap: 10px, column-gap: 20px */
}

/* Propiedades individuales */
.grid-gap-individual {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  row-gap: 15px;     /* Espacio entre filas */
  column-gap: 25px;  /* Espacio entre columnas */
}

/* Sin gap (comportamiento por defecto) */
.grid-no-gap {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 0;  /* Elementos pegados */
}

grid-area: posicionamiento específico

Permite posicionar elementos en lugares específicos del grid:

/* Sintaxis: grid-area: row-start / column-start / row-end / column-end */

.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 100px);
  gap: 10px;
}

/* Posicionamiento específico */
.item-1 {
  grid-area: 1 / 1 / 2 / 3;  /* Fila 1, Columnas 1-2 */
  background-color: #ff6b6b;
}

.item-2 {
  grid-area: 1 / 3 / 3 / 5;  /* Filas 1-2, Columnas 3-4 */
  background-color: #4ecdc4;
}

.item-3 {
  grid-area: 2 / 1 / 4 / 2;  /* Filas 2-3, Columna 1 */
  background-color: #45b7d1;
}

/* Resultado: elementos posicionados específicamente, 
   sin importar el orden en HTML */

Propiedades de línea individual

Control granular de posicionamiento:

/* Propiedades individuales en lugar de grid-area */
.positioned-item {
  grid-column-start: 2;    /* Comienza en columna 2 */
  grid-column-end: 4;      /* Termina en columna 4 */
  grid-row-start: 1;       /* Comienza en fila 1 */
  grid-row-end: 3;         /* Termina en fila 3 */
}

/* Shorthand para columnas y filas */
.shorthand-item {
  grid-column: 2 / 4;      /* = grid-column-start: 2; grid-column-end: 4; */
  grid-row: 1 / 3;         /* = grid-row-start: 1; grid-row-end: 3; */
}

/* Usando span para tamaño relativo */
.span-item {
  grid-column: 1 / span 2;  /* Comienza en 1, abarca 2 columnas */
  grid-row: 2 / span 1;     /* Comienza en 2, abarca 1 fila */
}

/* Usar líneas negativas (desde el final) */
.negative-lines {
  grid-column: 1 / -1;      /* Desde primera hasta última columna */
  grid-row: -2 / -1;        /* Última fila */
}

Funciones avanzadas para dimensiones

minmax() - tamaño mínimo y máximo

/* Columnas que se adaptan pero con límites */
.grid-minmax {
  display: grid;
  grid-template-columns: 
    minmax(200px, 300px)    /* Min 200px, Max 300px */
    1fr                     /* Resto del espacio */
    minmax(100px, 1fr);     /* Min 100px, Max flexible */
}

/* Filas responsivas */
.grid-minmax-rows {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 
    auto                    /* Altura del contenido */
    minmax(200px, 1fr)     /* Min 200px, resto flexible */
    auto;                   /* Altura del contenido */
}

fit-content() - ajuste al contenido

/* Ajusta al contenido pero con máximo */
.grid-fit-content {
  display: grid;
  grid-template-columns: 
    fit-content(200px)      /* Tamaño del contenido, max 200px */
    1fr                     /* Resto */
    fit-content(150px);     /* Tamaño del contenido, max 150px */
}

/* Útil para sidebars que se adaptan */
.sidebar-adaptive {
  display: grid;
  grid-template-columns: fit-content(250px) 1fr;
  /* Sidebar: tamaño del contenido, máximo 250px
     Contenido: resto del espacio */
}

auto-fit vs auto-fill

/* auto-fill: mantiene pistas vacías */
.grid-auto-fill {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  /* Crea tantas columnas de min 200px como quepan
     Deja espacios vacíos si sobra espacio */
}

/* auto-fit: colapsa pistas vacías */
.grid-auto-fit {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  /* Crea tantas columnas como elementos haya
     Los elementos se expanden para llenar el espacio */
}

/* Ejemplo visual con 3 elementos en contenedor ancho:
   auto-fill: [Item] [Item] [Item] [    ] [    ]
   auto-fit:  [  Item  ] [  Item  ] [  Item  ] */

Grid implícito vs explícito

/* Grid explícito: defines las pistas */
.explicit-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 columnas definidas */
  grid-template-rows: repeat(2, 100px);   /* 2 filas definidas */
  /* Grid explícito: 3x2 = 6 celdas */
}

/* Si agregas más de 6 elementos... */
.explicit-grid {
  /* Elementos 7, 8, 9... crean filas automáticas (grid implícito) */
  grid-auto-rows: 80px;        /* Altura de filas automáticas */
  grid-auto-flow: row;         /* Dirección de elementos adicionales */
}

/* Control del flujo automático */
.auto-flow-column {
  display: grid;
  grid-template-rows: repeat(3, 100px);
  grid-auto-columns: 150px;    /* Ancho de columnas automáticas */
  grid-auto-flow: column;      /* Elementos adicionales en nuevas columnas */
}

Diferencias con Flexbox

Aunque Grid y Flexbox pueden parecer similares, cada uno tiene casos de uso específicos y fortalezas diferentes.

Dimensionalidad: La diferencia clave

Flexbox - Unidimensional:

/* Flexbox controla UNA dimensión a la vez */
.flex-container {
  display: flex;
  /* Elementos se organizan en UNA dirección:
     - Horizontal (flex-direction: row)
     - Vertical (flex-direction: column) */
}

/* Ideal para: distribución lineal */
/* [Item1] → [Item2] → [Item3] → [Item4] */

Grid - Bidimensional:

/* Grid controla DOS dimensiones simultáneamente */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 1fr 1fr;
  /* Elementos se organizan en filas Y columnas */
}

/* Ideal para: layouts complejos 2D */
/* [Item1] [Item2]
   [Item3] [Item4] */

Cuándo usar cada uno

✅ Usa FLEXBOX cuando:

  • Distribución lineal: elementos en fila o columna
  • Componentes pequeños: navegación, botones, formularios
  • Alineación simple: centrar elementos
  • Distribución de espacio: space-between, space-around
  • Orden flexible: cambiar orden con order
  • Tamaño basado en contenido: elementos que crecen según contenido

Ejemplos ideales para Flexbox:

/* Navegación horizontal */
.nav-flex {
  display: flex;
  gap: 2rem;
  align-items: center;
}

/* Distribución de botones */
.button-group {
  display: flex;
  gap: 1rem;
  justify-content: flex-end;
}

/* Centrado perfecto */
.center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

/* Cards de igual altura */
.cards-flex {
  display: flex;
  gap: 1rem;
}

.card {
  flex: 1;  /* Distribución igual */
}

✅ Usa GRID cuando:

  • Layouts complejos: necesitas control 2D
  • Posicionamiento específico: elementos en lugares exactos
  • Superposición: elementos que se solapan
  • Layouts de página: header, sidebar, content, footer
  • Galerías asimétricas: elementos de diferentes tamaños
  • Responsive complejo: cambios dramáticos de layout

Ejemplos ideales para Grid:

/* Layout de página completo */
.page-layout {
  display: grid;
  grid-template-areas: 
    "header header"
    "sidebar content"
    "footer footer";
  grid-template-columns: 200px 1fr;
  grid-template-rows: auto 1fr auto;
}

/* Galería asimétrica */
.gallery-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 1rem;
}

.featured {
  grid-column: span 2;  /* Elemento destacado más ancho */
}

/* Dashboard con widgets */
.dashboard {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 1fr);
  gap: 1rem;
}

.widget-large {
  grid-column: span 2;
  grid-row: span 2;
}

Comparación práctica lado a lado

Caso: Centrar contenido

/* CON FLEXBOX - más simple para centrado */
.center-flexbox {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

/* CON GRID - más verboso para centrado simple */
.center-grid {
  display: grid;
  place-items: center;  /* shorthand para align-items + justify-items */
  height: 100vh;
}

/* GANADOR: Flexbox para centrado simple */

Caso: Layout de página

/* CON FLEXBOX - requiere nesting */
.page-flexbox {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

.main-flexbox {
  display: flex;  /* Nested flexbox */
  flex: 1;
}

.sidebar { flex: 0 0 200px; }
.content { flex: 1; }

/* CON GRID - layout directo, sin nesting */
.page-grid {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: auto 1fr auto;
  height: 100vh;
}

.header { grid-column: 1 / -1; }
.footer { grid-column: 1 / -1; }

/* GANADOR: Grid para layouts complejos */

Caso: Navegación responsiva

/* CON FLEXBOX - natural para navegación */
.nav-flexbox {
  display: flex;
  gap: 2rem;
  align-items: center;
}

@media (max-width: 768px) {
  .nav-flexbox {
    flex-direction: column;  /* Cambio simple */
  }
}

/* CON GRID - sobrecomplejo para navegación */
.nav-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
  gap: 2rem;
  align-items: center;
}

/* GANADOR: Flexbox para navegación */

Caso: Galería de imágenes

/* CON FLEXBOX - limitado para galerías */
.gallery-flexbox {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.gallery-flexbox .item {
  flex: 1 1 200px;  /* No controla filas específicas */
}

/* CON GRID - ideal para galerías */
.gallery-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 1rem;
}

.featured {
  grid-column: span 2;  /* Control específico */
  grid-row: span 2;
}

/* GANADOR: Grid para galerías complejas */

¿Se pueden combinar?

¡Absolutamente! Grid y Flexbox funcionan perfectamente juntos:

/* Grid para layout general, Flexbox para componentes */
.page-layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: auto 1fr auto;
  gap: 1rem;
}

/* Flexbox dentro de elementos grid */
.header {
  grid-column: 1 / -1;
  display: flex;              /* Flexbox dentro de Grid */
  justify-content: space-between;
  align-items: center;
}

.sidebar {
  display: flex;              /* Flexbox dentro de Grid */
  flex-direction: column;
  gap: 1rem;
}

.content .card-container {
  display: grid;              /* Grid dentro de Grid */
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
}

.card {
  display: flex;              /* Flexbox dentro de Grid */
  flex-direction: column;
}

.card .button-group {
  display: flex;              /* Flexbox dentro de Flexbox */
  gap: 1rem;
  margin-top: auto;
}

Resumen de decisión

Flowchart de decisión:

¿Necesitas layout en 2 dimensiones (filas Y columnas)?
├─ SÍ → Usa GRID
│  ├─ Layout de página completo
│  ├─ Posicionamiento específico
│  ├─ Galería asimétrica
│  └─ Dashboard/widgets
│
└─ NO → ¿Es distribución lineal (una dirección)?
   ├─ SÍ → Usa FLEXBOX
   │  ├─ Navegación
   │  ├─ Distribución de botones
   │  ├─ Centrado simple
   │  └─ Cards de igual altura
   │
   └─ ¿Necesitas compatibilidad antigua?
      ├─ SÍ → Usa floats + positioning
      └─ NO → Usa Grid o Flexbox según el caso

Ejercicio Práctico

Galería de imágenes con CSS Grid

Intermedio

Objetivo: Crear una galería de imágenes responsive usando CSS Grid que se adapte automáticamente al tamaño de pantalla.

Requisitos:

  1. Usar display: grid como base del layout
  2. Columnas que se adapten automáticamente (auto-fit)
  3. Tamaño mínimo de 250px por imagen
  4. Espaciado uniforme entre imágenes
  5. Algunas imágenes "destacadas" que ocupen más espacio
  6. Efectos hover en las imágenes
  7. Completamente responsive sin media queries

Estructura HTML sugerida:

<div class="gallery-container">
  <h2 class="gallery-title">Galería de Fotos</h2>
  
  <div class="image-gallery">
    <div class="gallery-item">
