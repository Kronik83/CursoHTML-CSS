Propiedades de transición (transition)

Las transiciones CSS permiten cambiar propiedades de manera gradual durante un período determinado, creando efectos suaves y profesionales.

Sintaxis básica

/* Transition simple */
.button {
  background-color: #007bff;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #0056b3;  /* Cambia suavemente en 0.3s */
}

/* Sintaxis completa */
transition: property duration timing-function delay;

/* Ejemplo:
   - property: propiedad a animar (background-color)
   - duration: duración (0.3s)
   - timing-function: curva de animación (ease)
   - delay: retraso antes de iniciar (0s) */

Propiedades de transition

transition-property

/* Transición de una propiedad */
.box {
  transition-property: transform;
}

/* Múltiples propiedades */
.box {
  transition-property: transform, opacity, background-color;
}

/* Todas las propiedades */
.box {
  transition-property: all;  /* Anima todo lo que cambie */
}

transition-duration

/* Duración en segundos */
.fast {
  transition-duration: 0.2s;   /* Rápida */
}

.normal {
  transition-duration: 0.3s;   /* Normal */
}

.slow {
  transition-duration: 0.5s;   /* Lenta */
}

/* Múltiples duraciones */
.multi {
  transition-property: transform, opacity;
  transition-duration: 0.3s, 0.5s;  /* transform: 0.3s, opacity: 0.5s */
}

transition-timing-function

/* Curvas de timing predefinidas */
.ease {
  transition-timing-function: ease;  /* Inicio lento, rápido, final lento */
}

.linear {
  transition-timing-function: linear;  /* Velocidad constante */
}

.ease-in {
  transition-timing-function: ease-in;  /* Inicio lento */
}

.ease-out {
  transition-timing-function: ease-out;  /* Final lento */
}

.ease-in-out {
  transition-timing-function: ease-in-out;  /* Inicio y final lentos */
}

/* Curvas personalizadas con cubic-bezier */
.custom {
  transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);
  /* Crea efectos de "rebote" u otros movimientos personalizados */
}

transition-delay

/* Retraso antes de iniciar */
.delayed {
  transition-delay: 0.2s;  /* Espera 0.2s antes de iniciar */
}

/* Retrasos escalonados */
.item:nth-child(1) { transition-delay: 0s; }
.item:nth-child(2) { transition-delay: 0.1s; }
.item:nth-child(3) { transition-delay: 0.2s; }
.item:nth-child(4) { transition-delay: 0.3s; }

Shorthand transition

/* Forma corta (recomendado) */
.element {
  transition: property duration timing-function delay;
}

/* Ejemplos prácticos */
.button {
  transition: background-color 0.3s ease;
}

.card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.menu-item {
  transition: all 0.2s ease-in-out;
}

Animaciones con @keyframes

@keyframes permite crear animaciones complejas con múltiples estados y control total sobre el proceso de animación.

Sintaxis básica

/* Definir la animación */
@keyframes nombreAnimacion {
  from {
    /* Estado inicial */
    opacity: 0;
  }
  to {
    /* Estado final */
    opacity: 1;
  }
}

/* Aplicar la animación */
.element {
  animation: nombreAnimacion 1s ease;
}

Keyframes con porcentajes

/* Control preciso con porcentajes */
@keyframes deslizar {
  0% {
    transform: translateX(-100%);
    opacity: 0;
  }
  50% {
    opacity: 1;
  }
  100% {
    transform: translateX(0);
    opacity: 1;
  }
}

/* Animación compleja */
@keyframes rebote {
  0%, 100% {
    transform: translateY(0);
  }
  25% {
    transform: translateY(-20px);
  }
  50% {
    transform: translateY(-10px);
  }
  75% {
    transform: translateY(-5px);
  }
}

Propiedades de animation

/* animation-name */
.element {
  animation-name: deslizar;
}

/* animation-duration */
.element {
  animation-duration: 2s;
}

/* animation-timing-function */
.element {
  animation-timing-function: ease-in-out;
}

/* animation-delay */
.element {
  animation-delay: 0.5s;
}

/* animation-iteration-count */
.element {
  animation-iteration-count: 3;        /* 3 veces */
  animation-iteration-count: infinite; /* Infinito */
}

/* animation-direction */
.element {
  animation-direction: normal;          /* Normal */
  animation-direction: reverse;         /* Al revés */
  animation-direction: alternate;       /* Alterna */
  animation-direction: alternate-reverse; /* Alterna inverso */
}

/* animation-fill-mode */
.element {
  animation-fill-mode: forwards;   /* Mantiene estado final */
  animation-fill-mode: backwards;  /* Aplica estado inicial */
  animation-fill-mode: both;       /* Ambos */
}

/* animation-play-state */
.element {
  animation-play-state: running;  /* Ejecutándose */
  animation-play-state: paused;   /* Pausada */
}

Shorthand animation

/* Sintaxis completa */
animation: name duration timing-function delay iteration-count direction fill-mode;

/* Ejemplos prácticos */

  animation: fadeIn 1s ease-in-out;
}

.spin-infinite {
  animation: spin 2s linear infinite;
}

.slide-and-fade {
  animation: slideIn 0.5s ease-out forwards;
}

3. Animaciones al scroll (con JavaScript)

HTML y CSS base

<div class="reveal">
  <h2>Contenido que aparece al hacer scroll</h2>
</div>

/* Estados de animación */
.reveal {
  opacity: 0;
  transform: translateY(50px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}

.reveal.active {
  opacity: 1;
  transform: translateY(0);
}

/* Variaciones */
.reveal-left {
  opacity: 0;
  transform: translateX(-50px);
}

.reveal-left.active {
  opacity: 1;
  transform: translateX(0);
}

JavaScript para detectar scroll

// Detectar elementos al hacer scroll
function revealOnScroll() {
  const reveals = document.querySelectorAll('.reveal');
  
  reveals.forEach(element => {
    const windowHeight = window.innerHeight;
    const elementTop = element.getBoundingClientRect().top;
    const elementVisible = 150;
    
    if (elementTop < windowHeight - elementVisible) {
      element.classList.add('active');
    }
  });
}

window.addEventListener('scroll', revealOnScroll);
revealOnScroll(); // Ejecutar al cargar

Performance y buenas prácticas

/* ✅ BUENAS PRÁCTICAS */

/* 1. Animar solo propiedades de bajo costo */
/* Económicas: transform, opacity */
.good-performance {
  transition: transform 0.3s, opacity 0.3s;
}

/* ❌ Costosas: width, height, top, left */
.bad-performance {
  transition: width 0.3s, height 0.3s;  /* Evitar */
}

/* 2. Usar will-change para animaciones complejas */
.optimized {
  will-change: transform, opacity;
  transition: transform 0.3s, opacity 0.3s;
}

/* 3. Limitar animaciones infinitas */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

/* 4. Usar transform en lugar de position */
/* ✅ Mejor */
.slide {
  transform: translateX(100px);
}

/* ❌ Peor */
.slide-bad {
  left: 100px;  /* Causa reflow */
}

/* 5. GPU acceleration */
.accelerated {
  transform: translateZ(0);  /* Fuerza aceleración por GPU */
}

Ejercicio Práctico

Animación de entrada al cargar y efecto hover animado

Intermedio-Avanzado

Objetivo: Crear una landing page con animaciones de entrada al cargar la página y múltiples efectos hover en elementos interactivos.

Requisitos:

  1. Hero section con animación de fade-in + slide-up
  2. Título con efecto de escritura (typewriter)
  3. Tarjetas de servicios con entrada escalonada
  4. Botones con efectos hover avanzados (ripple, scale, etc.)
  5. Animaciones al hacer scroll (reveal on scroll)
  6. Loading spinner mientras "carga" contenido
  7. Uso de @keyframes y transitions

Animaciones a implementar:

  • ✅ Fade-in del hero: 1s ease-out
  • ✅ Slide-up de elementos: translateY(50px) → translateY(0)
  • ✅ Entrada escalonada: delays de 0.1s, 0.2s, 0.3s
  • ✅ Hover en botones: scale(1.05) + box-shadow
  • ✅ Hover en cards: translateY(-10px) + sombra
  • ✅ Spinner infinito mientras carga
  • ✅ Reveal on scroll con IntersectionObserver

Código JavaScript para scroll animations:

// Intersection Observer para animaciones al scroll
const observerOptions = {
  threshold: 0.1,
  rootMargin: '0px 0px -100px 0px'
};

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('active');
    }
  });
}, observerOptions);

document.querySelectorAll('.reveal').forEach(el => {
  observer.observe(el);
});

Bonus (opcional):

  • ⭐ Parallax effect en el hero
  • ⭐ Gradiente animado en el background
  • ⭐ Contador animado (números que suben)
  • ⭐ Respeta prefers-reduced-motion
  • ⭐ Partículas flotantes decorativas
