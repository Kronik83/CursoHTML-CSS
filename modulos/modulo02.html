Etiquetas de texto

Las etiquetas de texto son fundamentales en HTML para estructurar y dar formato al contenido textual de una página web. Cada etiqueta tiene un propósito específico y un significado semántico.

Encabezados (h1-h6)

Los encabezados definen la jerarquía del contenido, desde el más importante (h1) hasta el menos importante (h6):

Ejemplo de encabezados:

<h1>Título principal</h1>
<h2>Subtítulo importante</h2>
<h3>Sección</h3>
<h4>Subsección</h4>
<h5>Apartado</h5>
<h6>Subapartado</h6>

Párrafos y saltos de línea

  • <p>: Define un párrafo de texto
  • <br>: Inserta un salto de línea
<p>Este es un párrafo completo con texto.</p>
<p>Este es otro párrafo.<br>
Con un salto de línea en el medio.</p>

Etiquetas de formato

  • <strong>: Texto importante (negrita semántica)
  • <em>: Texto enfatizado (cursiva semántica)
  • <u>: Texto subrayado
  • <small>: Texto pequeño (letra pequeña)
  • <mark>: Texto resaltado

Ejemplo de formato:

<p>Este texto tiene <strong>palabras importantes</strong> y 
<em>texto enfatizado</em>. También podemos <u>subrayar</u> 
y <mark>resaltar</mark> texto.</p>
<small>Texto en letra pequeña para notas.</small>

Buenas prácticas

  • Usa solo un <h1> por página
  • Mantén la jerarquía lógica de encabezados
  • Prefiere <strong> y <em> sobre <b> e <i>
  • No uses <br> para espaciado, usa CSS

Listas en HTML

Las listas son elementos fundamentales para organizar información de manera estructurada. HTML ofrece diferentes tipos de listas según el propósito.

Listas no ordenadas (<ul>)

Se usan cuando el orden de los elementos no es importante. Los elementos se marcan con viñetas.

Ejemplo de lista no ordenada:

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

Listas ordenadas (<ol>)

Se usan cuando el orden de los elementos es importante. Los elementos se numeran automáticamente.

Ejemplo de lista ordenada:

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
    <li>Tercer paso</li>
</ol>

Listas anidadas

Puedes crear listas dentro de otras listas para estructura jerárquica:

Ejemplo de listas anidadas:

<ul>
    <li>Frutas
        <ul>
            <li>Manzanas</li>
            <li>Naranjas</li>
        </ul>
    </li>
    <li>Verduras
        <ul>
            <li>Lechuga</li>
            <li>Tomate</li>
        </ul>
    </li>
</ul>

Atributos útiles

  • start: Define el número inicial en listas ordenadas
  • type: Cambia el tipo de numeración (1, A, a, I, i)
  • reversed: Invierte el orden de numeración
<ol start="5" type="A">
    <li>Elemento E</li>
    <li>Elemento F</li>
</ol>

Enlaces en HTML

Los enlaces son la base de la navegación web y permiten conectar páginas, secciones y recursos. La etiqueta <a> es fundamental para crear hipervínculos.

Enlaces externos

Conectan a páginas web externas usando URLs completas:

Ejemplo de enlace externo:

<a href="https://www.google.com">Ir a Google</a>
<a href="https://www.github.com" target="_blank">GitHub (nueva pestaña)</a>

Enlaces internos

Navegan dentro del mismo sitio web usando rutas relativas:

Ejemplo de enlaces internos:

<a href="pagina2.html">Ir a página 2</a>
<a href="../index.html">Volver al inicio</a>
<a href="contacto.html">Contacto</a>

Enlaces de anclaje

Navegan a secciones específicas dentro de la misma página:

Ejemplo de enlaces de anclaje:

<a href="#seccion1">Ir a sección 1</a>
<a href="#top">Ir arriba</a>

<!-- Destino del enlace -->
<h2 id="seccion1">Sección 1</h2>

Enlaces especiales

  • Email: <a href="mailto:correo@ejemplo.com">
  • Teléfono: <a href="tel:+123456789">
  • Descarga: <a href="archivo.pdf" download>
<a href="mailto:contacto@ejemplo.com">Enviar email</a>
<a href="tel:+34123456789">Llamar</a>
<a href="documento.pdf" download>Descargar PDF</a>

Atributos importantes

  • target="_blank": Abre en nueva pestaña
  • title: Texto de ayuda al pasar el mouse
  • download: Fuerza la descarga del archivo
  • rel="noopener": Seguridad para enlaces externos

Citas en HTML

Las citas son elementos importantes para incluir referencias, testimonios o fragmentos de texto de otras fuentes. HTML proporciona etiquetas específicas para diferentes tipos de citas.

Citas en bloque (<blockquote>)

Se usan para citas largas que ocupan su propio párrafo, generalmente con sangría:

Ejemplo de blockquote:

<blockquote cite="https://fuente.com">
    <p>La imaginación es más importante que el conocimiento. 
    El conocimiento es limitado, mientras que la imaginación 
    abarca el mundo entero.</p>
    <footer>— Albert Einstein</footer>
</blockquote>

Citas en línea (<q>)

Se usan para citas cortas dentro de un párrafo. El navegador añade comillas automáticamente:

Ejemplo de cita en línea:

<p>Como dijo Gandhi: <q>Sé el cambio que quieres ver en el mundo</q>.</p>

Elemento <cite>

Se usa para referenciar el título de una obra (libro, película, artículo, etc.):

Ejemplo con cite:

<p>Mi libro favorito es <cite>Cien años de soledad</cite> 
de Gabriel García Márquez.</p>

Atributo cite

Proporciona la URL de la fuente original de la cita:

<blockquote cite="https://www.ejemplo.com/articulo">
    <p>Contenido de la cita aquí.</p>
</blockquote>

Mejores prácticas

  • Usa <blockquote> para citas largas (más de una línea)
  • Usa <q> para citas cortas dentro del texto
  • Incluye siempre la fuente de la cita
  • Usa el atributo cite para referenciar la fuente
  • Considera usar <footer> dentro de blockquote para la atribución

Ejemplo completo:

<article>
    <h2>Sobre la creatividad</h2>
    <p>Muchos pensadores han reflexionado sobre la creatividad.</p>
    
    <blockquote cite="https://ejemplo.com">
        <p>La creatividad es la inteligencia divirtiéndose.</p>
        <footer>— <cite>Albert Einstein</cite></footer>
    </blockquote>
    
    <p>Esta idea se relaciona con lo que Jobs decía: 
    <q>La innovación distingue a un líder de un seguidor</q>.</p>
</article>

Ejercicio Práctico

Sección de presentación personal

Principiante

Objetivo: Agrega una sección de presentación personal con encabezado, párrafos, una lista de intereses y un enlace a una red social.

Pasos a seguir:

  1. Crea un encabezado principal con tu nombre
  2. Agrega párrafos describiendo tu experiencia
  3. Incluye una lista de tus intereses
  4. Añade un enlace a tu perfil de red social
  5. Usa etiquetas de formato para resaltar información

Ejemplo de código:

<section>
    <h1>Juan Pérez</h1>
    <p>Soy un <strong>desarrollador web</strong> con pasión por crear 
       experiencias digitales <em>increíbles</em>.</p>
    
    <h2>Mis intereses</h2>
    <ul>
        <li>Desarrollo frontend</li>
        <li>Diseño UX/UI</li>
        <li>Tecnologías emergentes</li>
    </ul>
    
    <p>Puedes encontrarme en 
       <a href="https://linkedin.com/in/juanperez">LinkedIn</a>
    </p>
</section>
