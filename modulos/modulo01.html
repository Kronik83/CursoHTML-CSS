Qué es HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Es la columna vertebral de todo contenido web y define la estructura y el significado del contenido web mediante el uso de etiquetas.

¿Qué significa HTML?

  • HyperText: Se refiere a los enlaces que conectan páginas web entre sí
  • Markup: Significa "marcado", refiriéndose a las etiquetas que estructuran el contenido
  • Language: Es un lenguaje con su propia sintaxis y reglas

Función de HTML en la web

HTML no es un lenguaje de programación, sino un lenguaje de marcado que:

  • Define la estructura del contenido (encabezados, párrafos, listas, etc.)
  • Proporciona significado semántico al contenido
  • Permite la inclusión de multimedia (imágenes, videos, audio)
  • Facilita la navegación mediante enlaces
  • Sirve como base para aplicar estilos CSS y funcionalidad JavaScript

Ejemplo básico de HTML:

<h1>Este es un encabezado</h1>
<p>Este es un párrafo de texto.</p>
<a href="https://ejemplo.com">Este es un enlace</a>

Estructura básica de un documento HTML

Todo documento HTML debe seguir una estructura específica que incluye elementos fundamentales. Esta estructura asegura que el navegador interprete correctamente el contenido.

Componentes esenciales:

1. DOCTYPE

Declara el tipo de documento y la versión de HTML que se está utilizando.

<!DOCTYPE html>

2. Elemento <html>

Es el elemento raíz que envuelve todo el contenido de la página.

<html lang="es">
    <!-- Todo el contenido va aquí -->
</html>

3. Elemento <head>

Contiene metadatos sobre el documento que no se muestran en la página.

  • <title>: Título que aparece en la pestaña del navegador
  • <meta charset="UTF-8">: Codificación de caracteres
  • <meta name="viewport">: Configuración para dispositivos móviles

4. Elemento <body>

Contiene todo el contenido visible de la página web.

Estructura completa:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    <!-- Contenido visible aquí -->
</body>
</html>

Etiquetas y atributos

Las etiquetas HTML son los bloques de construcción básicos que definen la estructura y el contenido de una página web.

¿Qué son las etiquetas?

Las etiquetas HTML están rodeadas por corchetes angulares <> y generalmente vienen en pares:

  • Etiqueta de apertura: <etiqueta>
  • Etiqueta de cierre: </etiqueta>
  • Contenido: Todo lo que está entre las etiquetas

Ejemplo de etiqueta:

<p>Este es el contenido del párrafo</p>

Etiquetas auto-cerradas

Algunas etiquetas no necesitan etiqueta de cierre:

<br>     <!-- Salto de línea -->
<img>    <!-- Imagen -->
<hr>     <!-- Línea horizontal -->
<input>  <!-- Campo de entrada -->

¿Qué son los atributos?

Los atributos proporcionan información adicional sobre los elementos HTML:

  • Se escriben dentro de la etiqueta de apertura
  • Tienen la forma nombre="valor"
  • Modifican el comportamiento o apariencia del elemento

Ejemplos de atributos:

<a href="https://ejemplo.com" target="_blank">Enlace</a>
<img src="imagen.jpg" alt="Descripción de la imagen">
<p class="destacado" id="parrafo1">Texto</p>

Jerarquía de elementos

Los elementos HTML pueden anidarse (contener otros elementos), creando una estructura jerárquica:

<div>
    <h2>Título de sección</h2>
    <p>Este párrafo está <strong>dentro</strong> de un div.</p>
    <ul>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
    </ul>
</div>

Comentarios en HTML

Los comentarios en HTML son notas que puedes agregar a tu código para explicar qué hace una sección específica. Los comentarios no se muestran en el navegador.

Sintaxis de comentarios

Los comentarios HTML comienzan con <!-- y terminan con -->:

<!-- Este es un comentario -->

Usos de los comentarios

  • Documentación: Explicar secciones complejas del código
  • Notas temporales: Recordatorios para futuras modificaciones
  • Deshabilitar código: "Comentar" código sin eliminarlo
  • Organización: Marcar inicio y fin de secciones

Ejemplos prácticos:

<!-- Inicio de la sección del encabezado -->
<header>
    <h1>Mi sitio web</h1>
    <!-- TODO: Agregar menú de navegación -->
</header>
<!-- Fin de la sección del encabezado -->

<!-- 
Este es un comentario de múltiples líneas
que puede extenderse por varias líneas
para explicar código complejo
-->

Buenas prácticas

  • Usa comentarios para explicar el "por qué", no el "qué"
  • Mantén los comentarios actualizados con el código
  • No abuses de los comentarios; el código claro es mejor que muchos comentarios
  • Usa comentarios para marcar secciones importantes

Editor de código: Visual Studio Code

Visual Studio Code (VS Code) es el editor de código más popular para desarrollo web. Es gratuito, potente y tiene excelente soporte para HTML, CSS y JavaScript.

¿Por qué VS Code?

  • Gratuito y código abierto: Disponible para Windows, Mac y Linux
  • Intellisense: Autocompletado inteligente de código
  • Extensiones: Miles de extensiones para mejorar la productividad
  • Terminal integrado: No necesitas salir del editor
  • Control de versiones: Git integrado
  • Live Server: Previsualización en tiempo real

Configuración inicial

1. Instalación

Descarga VS Code desde code.visualstudio.com

2. Extensiones recomendadas para HTML/CSS

  • Live Server: Servidor local con recarga automática
  • Auto Rename Tag: Renombra etiquetas automáticamente
  • HTML CSS Support: Autocompletado mejorado
  • Prettier: Formateador de código
  • Bracket Pair Colorizer: Colorea las llaves y paréntesis

3. Atajos de teclado útiles

  • Ctrl + /: Comentar/descomentar línea
  • Ctrl + D: Seleccionar siguiente ocurrencia
  • Alt + ↑/↓: Mover línea arriba/abajo
  • Ctrl + Shift + P: Paleta de comandos
  • Ctrl + `: Abrir/cerrar terminal

Creando tu primer archivo HTML

  1. Abre VS Code
  2. Crea una nueva carpeta para tu proyecto
  3. Abre la carpeta en VS Code (File → Open Folder)
  4. Crea un nuevo archivo: index.html
  5. Escribe ! y presiona Tab para generar la estructura básica
  6. Instala Live Server y haz clic derecho → "Open with Live Server"

Truco: Emmet en VS Code

Ejercicio Práctico

Crear tu primer documento HTML

Principiante

Objetivo: Crear un archivo index.html con la estructura básica del documento e incluir un título y un párrafo de bienvenida.

Pasos a seguir:

  1. Crea un nuevo archivo llamado index.html
  2. Agrega la declaración DOCTYPE
  3. Estructura las etiquetas html, head y body
  4. Incluye un título en la etiqueta title
  5. Agrega un párrafo de bienvenida en el body

Ejemplo de código:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primera página web</title>
</head>
<body>
    <h1>¡Bienvenido a mi primera página!</h1>
    <p>Este es mi primer documento HTML.</p>
</body>
</html>
