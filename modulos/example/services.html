Ejemplos de Servicios para el Proyecto Final

Partiendo de lo aprendido se muestran diferentes estilos de Seccion de Servicios.

Sección de Servicios en grilla de Tarjetas.

El estilo de esta sección utiliza un diseño de cuadrícula para organizar las tarjetas de servicio de manera eficiente y atractiva. Las tarjetas no tienen un tamaño fijo, lo que permite que se adapten al espacio disponible. Tampoco tiene algún tipo de alineación.

Desarrollo Web

Creación de sitios web responsivos y funcionales.

Aplicaciones Móviles

Desarrollo de aplicaciones para iOS y Android.

SEO

Optimización para motores de búsqueda.

Responsive

Optimización para dispositivos móviles.

        <i class="fas fa-mobile-alt service-icon"></i>

El estilo de esta sección utiliza un diseño de Flexbox para mostrar las tarjetas de servicio, ubicando las tarjetas según el ancho del contenedor. Las tarjetas tienen un tamaño fijo.
