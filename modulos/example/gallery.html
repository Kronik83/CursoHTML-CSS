Ejemplos de Sección Galería para el Proyecto Final

Partiendo de lo aprendido se muestran diferentes estilos de Sección Galería.

Imagenes Reales:

Galería de Imágenes con Tamaño Original (Masonry).

Esta galería muestra las imágenes respetando sus proporciones y tamaños originales, organizándolas de forma dinámica usando CSS Grid con auto-flow dense.

Las imágenes se adaptan naturalmente sin forzar un tamaño específico, creando un efecto tipo Pinterest.

HTML:

<section class="gallery gallery-masonry">
    <div class="gallery-item-masonry"> 
        <img src="imagen1.jpg" alt="Descripción">
    </div>
    <div class="gallery-item-masonry"> 
        <img src="imagen2.jpg" alt="Descripción">
    </div>
    <!-- Más imágenes... -->
</section>

CSS:

.gallery-masonry {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
    grid-auto-rows: 10px; /* Filas pequeñas para mejor control */
    gap: 15px;
    padding: 15px;
}

.gallery-item-masonry {
    overflow: hidden;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    transition: transform 0.3s ease;
}

.gallery-item-masonry:hover {
    transform: translateY(-5px);
    box-shadow: 0 8px 12px rgba(0, 0, 0, 0.2);
}

.gallery-item-masonry img {
    width: 100%;
    height: auto; /* Mantiene la proporción original */
    display: block;
    object-fit: cover;
}

Galería Masonry con CSS Columns (Diseño Alternativo).

Este diseño utiliza CSS Columns para crear un efecto masonry fluido donde las imágenes se distribuyen automáticamente en columnas verticales.

Las imágenes fluyen de arriba hacia abajo en cada columna, creando un diseño de revista tipo Pinterest.

HTML:

<section class="gallery-columns">
    <div class="gallery-item-column"> 
        <img src="imagen1.jpg" alt="Descripción">
    </div>
    <div class="gallery-item-column"> 
        <img src="imagen2.jpg" alt="Descripción">
    </div>
    <!-- Más imágenes... -->
</section>

CSS:

.gallery-columns {
    column-count: 3; /* Número de columnas */
    column-gap: 15px; /* Espacio entre columnas */
    padding: 15px;
}

.gallery-item-column {
    break-inside: avoid; /* Evita que los items se corten entre columnas */
    margin-bottom: 15px;
    overflow: hidden;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    transition: all 0.3s ease;
}

.gallery-item-column:hover {
    transform: translateY(-5px) scale(1.02);
    box-shadow: 0 8px 16px rgba(0, 0, 0, 0.2);
}

.gallery-item-column img {
    width: 100%;
    height: auto; /* Mantiene proporción original */
    display: block;
}

/* Responsive */
@media (max-width: 768px) {
    .gallery-columns {
        column-count: 2; /* 2 columnas en tablets */
    }
}

@media (max-width: 480px) {
    .gallery-columns {
        column-count: 1; /* 1 columna en móviles */
    }
}
