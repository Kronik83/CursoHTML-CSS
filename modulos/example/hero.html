Ejemplos de HERO'S para el Proyecto Final

Partiendo de lo aprendido se muestran diferentes estilos de HERO'S.

Hero Simple con Texto Centrado

Bienvenido a Nuestro Sitio Web

Descubre nuestros servicios y productos.

Explorar Ahora

HTML:

<section class="hero hero-simple-centered">
    <div class="hero-content">
        <h1>Bienvenido a Nuestro Sitio Web</h1>
        <p>Descubre nuestros servicios y productos.</p>
        <a href="#" class="btn btn-primary btn-sm">Explorar Ahora</a>
    </div>

CSS:

.hero-simple-centered {
    background-color: #f5f5f5;
    display: flex;
    align-items: center;
    justify-content: center;
    height: 400px; /* Ajusta la altura según sea necesario */
    /*width: 100%;*/ /* Asegura que el hero ocupe todo el ancho disponible */
    text-align: center;
}

.hero-simple-centered h1 {
    font-size: 2.5rem;
    margin-bottom: 10px;
}

.hero-simple-centered p {
    font-size: 1.2rem;
    margin-bottom: 20px;
}

.hero-simple-centered .btn {
    font-size: 1rem;
    padding: 10px 20px;
}
