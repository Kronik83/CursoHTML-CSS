Ejemplos de headers para el Proyecto Final

Partiendo de lo aprendido se muestran diferentes estilos de header.

A tener en cuenta para el Encabezado.

Generalmente los encabezados contienen el logo y el menú de navegación.

Algunos incluyen otro menú para con los botones de login y registro, que cambia su contenido de acuerdo al usuario logueado.

Algunos tienen un slider en la parte superior o inferior, que utilizan para mostrar contenido destacado

Otros incluyen un buscador o un botón de llamada a la acción.

También hay otros que tienen un hero en la parte inferior.

Los diseños pueden variar según las especificaciones del cliente, pero en general deben ser atractivos y funcionales.

Tip:

Recuerda que el header es lo primero que ve un visitante al entrar a un sitio web, por lo que es importante causar una buena impresión.

Y si este tiene el menú principal de navegación, debe ser claro, fácil de usar y generalmente debería estar siempre presente.

Usando position:fixed; en los estilo puedes hacerlo siempre visible. en la parte donde nosotros lo coloquemos.

HTML:

<header>
    <div class="logo">
        <img src="logo.png" alt="Mi Logo">
    </div>
    <nav class="main-nav">
        <ul>
            <li><a href="#home">Inicio</a></li>
            <li><a href="#about">Sobre Mí</a></li>
            <li><a href="#services">Servicios</a></li>
            <li><a href="#gallery">Galería</a></li>
            <li><a href="#contact">Contacto</a></li>
        </ul>
    </nav>
</header>

CSS:

header {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
    background-color: #333;
    color: #fff;
}

.logo img {
    max-width: 100px;
}

.main-nav ul {
    list-style: none;
    padding: 0;
}

.main-nav li {
    display: inline;
    margin-right: 1rem;
}

.main-nav a {
    color: #fff;
    text-decoration: none;
}

.main-nav a:hover {
    background-color: #ffffff20;
    border-radius: 5px;
}
