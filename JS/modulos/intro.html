¿Que es JavaScript y para que sirve?

JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en páginas web.

¿Para qué se utiliza JavaScript?

JavaScript se utiliza para crear interactividad en las páginas web, como formularios dinámicos, animaciones y contenido actualizado en tiempo real.

Ejemplos de uso de JavaScript

Algunos ejemplos de lo que se puede hacer con JavaScript incluyen:

  • Validación de formularios
  • Creación de gráficos y visualizaciones
  • Desarrollo de juegos en línea
  • Interacción con APIs externas
  • En definitiva, darle vida a las páginas web.

Ejemplo básico de JavaScript

// Imprimir un mensaje en la consola
console.log("¡Hola, mundo!");
// Sumar dos números y mostrar el resultado
let a = 1;
let b = 2;
console.log(a + b);
// Funciones
function saludar(nombre) {
    return "Hola, " + nombre + "!";
}

let resultado = saludar("Juan");
console.log(resultado);

Historia y evolución del lenguaje

JavaScript fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications Corporation. Originalmente se llamó Mocha, luego LiveScript, y finalmente JavaScript para aprovechar la popularidad de Java en ese momento.

Evolución de JavaScript

Desde su creación, JavaScript ha evolucionado significativamente. Algunas de las etapas clave en su evolución incluyen:

  • 1996: Publicación de la primera especificación oficial, ECMAScript.
  • 2005: Popularización de AJAX, que permitió la creación de aplicaciones web más interactivas.
  • 2009: Lanzamiento de Node.js, que permitió ejecutar JavaScript en el servidor.
  • 2015: Publicación de ECMAScript 6 (ES6), que introdujo muchas características nuevas como clases, módulos y promesas.

Importancia actual de JavaScript

Hoy en día, JavaScript es uno de los lenguajes de programación más populares y ampliamente utilizados en el desarrollo web. Es esencial para crear experiencias web modernas y dinámicas.

¿Cómo se ejecuta JavaScript en el navegador?

JavaScript se ejecuta en el navegador a través del motor de JavaScript, que interpreta y ejecuta el código. Esto permite que las páginas web sean interactivas y dinámicas.

Proceso de ejecución

Cuando un navegador carga una página web, sigue estos pasos para ejecutar JavaScript:

  1. El navegador descarga el archivo HTML y analiza su contenido.
  2. Cuando encuentra una etiqueta <script>, descarga y ejecuta el código JavaScript contenido en ella o en un archivo externo.
  3. El motor de JavaScript interpreta el código línea por línea y realiza las acciones especificadas, como manipular el DOM o manejar eventos.

Motores de JavaScript populares

Algunos de los motores de JavaScript más populares incluyen:

  • V8: Utilizado por Google Chrome y Node.js.
  • SpiderMonkey: Utilizado por Mozilla Firefox.
  • JavaScriptCore: Utilizado por Safari.

¿Cómo se integra JavaScript con HTML y CSS?

JavaScript se puede integrar en HTML mediante la etiqueta <script> y puede manipular el DOM para cambiar el contenido y estilo de las páginas web.

Integración básica

Para incluir JavaScript en una página HTML, se utiliza la etiqueta <script>.

Puede colocarse en el <head>, este se ejecutará antes de que se cargue el contenido de la página. Generalmente se coloca la ruta de ubicación del archivo externo de JS.

O al final del <body> para mejorar el rendimiento. Aqui se puede colocar un archivo externo o colocar directamente el código JavaScript, de forma inline.

<!DOCTYPE html>
<html>
    <head>
        <title>Mi Página</title>
        <script src="script.js"></script> (Ubicación del archivo externo de JS)
    </head>
    <body>
        <h1>Hola, mundo!</h1>
        <script> (Codigo JS inline)
            console.log("Hola desde el script inline");
        </script>
    </body>
</html>

Primer Script: console.log()

El primer script que muchos programadores escriben es uno que utiliza console.log() para imprimir mensajes en la consola del navegador. Esto es útil para depurar y entender el flujo de ejecución del código.

// Imprimir un mensaje en la consola
console.log("Hola, mundo!");

Recuerda que debes tener el script vinculado correctamente en tu archivo HTML para que funcione o en línea.

Para ver el los mensajes de los console.log(), abre las herramientas de desarrollo de tu navegador (generalmente con F12 o Ctrl+Shift+I) y ve a la pestaña "Consola".

Comentarios y buenas prácticas básicas

Es importante escribir código limpio y bien comentado. Los comentarios ayudan a otros desarrolladores (y a ti mismo en el futuro) a entender el propósito y funcionamiento del código.

Tipos de comentarios en JavaScript

En JavaScript, existen dos tipos principales de comentarios:

  • Comentarios de una sola línea: Se utilizan para comentarios breves y se indican con //.
  • Comentarios de múltiples líneas: Se utilizan para comentarios más extensos y se encierran entre /* y */.
// Este es un comentario de una sola línea.
/*
  Este es un comentario
    de múltiples líneas.
*/

Buenas prácticas para comentarios

  • Escribe comentarios claros y concisos que expliquen el "por qué" del código, no solo el "qué".
  • Evita comentarios obvios que no aporten valor.
  • Mantén los comentarios actualizados cuando realices cambios en el código.
  • Utiliza comentarios para dividir el código en secciones lógicas.
