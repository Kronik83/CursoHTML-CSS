Variables

Tipos de variables:

Las variables en JavaScript son contenedores para almacenar datos. Se pueden declarar utilizando las palabras clave var, let o const.

Variables var y let

La palabra clave var como let se utiliza para declarar variables que toma valores que pueden cambiar a lo largo del tiempo. Solo que tienen un diferentes alcance (scope).

Variable const

La palabra const se utiliza para declarar variables cuyo valor no puede ser reasignado después de su inicialización.

¿Que pasa si despues de asignarle un valor intento cambiarlo?

Si intentas cambiar el valor de una variable declarada con const, obtendrás un error de tipo "TypeError".

Alcance de la variable var

Las variables declaradas con var tienen un alcance dentro del contexto de la función, lo que significa que son accesibles dentro de la función en la que se declaran.

¿Que pasa si llamo a esta variable fuera de la función?

Si intentas acceder a una variable declarada con var fuera de su función, obtendrás un error de referencia, ya que la variable no está definida en ese contexto.

Alcance de la variable let y const

Las variables declaradas con let y const tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque en el que se declaran.

¿Que pasa si llamo a esta variable fuera del bloque declarado?

Si intentas acceder a una variable declarada con let o const fuera del bloque en el que se definió, obtendrás un error de referencia, ya que la variable no está definida en ese contexto.

Ejemplo de variable var:

function ejemploVar() {
    var contador = 0;
    if (true) {
        var contador = 1; // Misma variable dentro del bloque
        console.log(contador); // Salida: 1
    }
    console.log(contador); // Salida: 1
}

Ejemplo de variable let:


function ejemploLet() {
    let contador = 0;  ────────────────────────────────────────────────┐
    if (true) {                                                        │
        let contador = 1; // Variable diferente dentro del bloque      │
        console.log(contador); // Salida: 1                            │
    }                                                                  │
    console.log(contador); // Salida: 0  < ────────────────────────────┘
}

Ejemplo de variable const:


function ejemploConst() {
    const pi = 3.14159;
    console.log(pi); // Salida: 3.14159
    
    pi = 0; // Error: Assignment to constant variable.
}

Tipos de Datos

Tipos de datos primitivos:

JavaScript tiene varios tipos de datos primitivos, incluyendo:

  • Número: Representa tanto enteros como números de punto flotante. Ejemplo: 42, 3.14
  • Cadena de texto (String): Representa secuencias de caracteres. Ejemplo: "Hola, Mundo!"
  • Boleano (Boolean): Representa valores lógicos: true o false.
  • Indefinido (Undefined): Indica que una variable no ha sido asignada a un valor.
  • Nulo (Null): Representa la ausencia intencional de cualquier valor.
  • Símbolo (Symbol): Representa un identificador único e inmutable.

Tipos de datos complejos:

Además de los tipos primitivos, JavaScript también tiene tipos de datos complejos, como:

  • Objeto (Object): Una colección de propiedades y métodos. Ejemplo: { nombre: "Juan", edad: 30 }
  • Matriz (Array): Una lista ordenada de valores. Ejemplo: [1, 2, 3, 4, 5]
  • Función (Function): Un bloque reutilizable de código que realiza una tarea específica.

Declaracion de variables segun su tipo:

Ejemplos de declaración de variables:

// Declaración de variables
var nombre = "Juan";  // String
let edad = 30;        // Number
const esEstudiante = true; // Boolean
let direccion;       // Undefined
const telefono = null; // Null
let simbolo = Symbol("id"); // Symbol

// Declaración de un objeto
let persona = {
    nombre: "Ana",
    edad: 25
};
// Declaración de un array
let numeros = [1, 2, 3, 4, 5];
// Declaración de una función
function saludar() {
    console.log("Hola, " + nombre + "!");
}

Operadores Aritméticos, Lógicos y de Comparación

Operadores Aritméticos:

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas. Algunos de los operadores aritméticos más comunes son:

  • + (Suma): Suma dos valores.
  • - (Resta): Resta un valor de otro.
  • * (Multiplicación): Multiplica dos valores.
  • / (División): Divide un valor entre otro.
  • % (Módulo): Devuelve el resto de una división.

Operadores Lógicos:

Los operadores lógicos se utilizan para combinar expresiones booleanas. Los operadores lógicos más comunes son:

  • && (AND): Devuelve true si ambas expresiones son verdaderas.
  • || (OR): Devuelve true si al menos una de las expresiones es verdadera.
  • ! (NOT): Invierte el valor booleano de una expresión.

Operadores de Comparación:

Los operadores de comparación se utilizan para comparar dos valores. Algunos de los operadores de comparación más comunes son:

  • == (Igualdad): Devuelve true si los valores son iguales (con conversión de tipo).
  • === (Identidad): Devuelve true si los valores y tipos son iguales.
  • != (Desigualdad): Devuelve true si los valores no son iguales (con conversión de tipo).
  • !== (No identidad): Devuelve true si los valores o tipos no son iguales.
  • > (Mayor que): Devuelve true si el valor izquierdo es mayor que el derecho.
  • < (Menor que): Devuelve true si el valor izquierdo es menor que el derecho.
  • >= (Mayor o igual que): Devuelve true si el valor izquierdo es mayor o igual que el derecho.
  • <= (Menor o igual que): Devuelve true si el valor izquierdo es menor o igual que el derecho.

Ejemplos de operadores:

// Operadores Aritméticos
let a = 5;
let b = 10;
let suma = a + b; // Salida 15
let resta = a - b; // Salida -5
let multiplicacion = a * b; // Salida 50
let division = a / b; // Salida 0.5
let modulo = a % b; // Salida 5

// Operadores Lógicos
let x = true;
let y = false;
let and = x && y; // Salida false
let or = x || y; // Salida true
let not = !x; // Salida false

// Operadores de Comparación
let a = 10;
let b = 21;
let igual = (a == b); // Salida false
let identidad = (a === b); // Salida false
let desigual = (a != b); // Salida true
let noIdentidad = (a !== b); // Salida true
let mayor = (a > b); // Salida false
let menor = (a < b); // Salida true
let mayorIgual = (a >= b); // Salida false
let menorIgual = (a <= b); // Salida true

Conversión de Tipos (Casting y Coerción)

Conversión de Tipos Explícita (Casting):

La conversión de tipos explícita, o casting, ocurre cuando el programador convierte deliberadamente un valor de un tipo a otro utilizando funciones o métodos específicos.

Funciones comunes para conversión explícita:

  • Number(): Convierte un valor a un número.
  • String(): Convierte un valor a una cadena de texto.
  • Boolean(): Convierte un valor a un booleano.

Conversión de Tipos Implícita (Coerción):

La conversión de tipos implícita, o coerción, ocurre automáticamente cuando JavaScript convierte un valor de un tipo a otro según el contexto en el que se utiliza.

Ejemplos comunes de conversión:

  • Cuando se utiliza el operador + con una cadena y un número, el número se convierte en cadena.
  • Cuando se utiliza un valor en un contexto booleano, JavaScript convierte el valor a true o false según su "truthiness".

Ejemplos de conversión de tipos:

// Conversión Explícita
let num = "5"; // Cadena los numero entre comillas dobles se consideran cadenas
let str = String(num); // Convierte a cadena
let numero = 0;
let numToString = String(numero); // Convierte a cadena
let strNum = "123";
let intNum = Number(strNum); // Convierte a número

let bool = Boolean(num); // Convierte a booleano Resultado: true
let boolZero = Boolean(numero); // Convierte a booleano Resultado: false

// Conversión Implícita
let resultado = "El número es: " + num; // Convierte a cadena

Estructuras Condicionales: if, else, switch

¿Que es una Condición?

Una condición es una expresión que se evalúa como verdadera o falsa. En JavaScript, las condiciones se utilizan en estructuras de control como if, else y switch.

Ejemplo de condición:

let edad = 18;
console.log(edad >= 18); // Salida: true
console.log(edad < 18);  // Salida: false

/* la condicion es una comparacion entre dos terminos que dan como resultado
un valor booleano (true o false).*/

Estructura if:

La estructura if se utiliza para ejecutar un bloque de código si una condición especificada es verdadera.

Sintaxis:

if (condición) {
    // Bloque de código a ejecutar si la condición es verdadera
}

Estructura if...else:

La estructura if...else permite ejecutar un bloque de código si la condición es verdadera y otro bloque si la condición es falsa.

Sintaxis:

if (condición) {
    // Bloque de código a ejecutar si la condición es verdadera
} else {
    // Bloque de código a ejecutar si la condición es falsa
}

Estructura switch:

La estructura switch se utiliza para ejecutar diferentes bloques de código según el valor de una expresión.

Sintaxis:

switch (expresión) {
case valor1:
    // Bloque de código a ejecutar si la expresión es igual a valor1
    break;
case valor2:
    // Bloque de código a ejecutar si la expresión es igual a valor2
    break;
default:
    // Bloque de código a ejecutar si la expresión no coincide con ningún valor
}

Ejemplos de Estructuras Condicionales

Ejemplo de if...else:

let edad = 20;
if (edad >= 18) { // resultado de la condicional es true
    console.log("Eres mayor de edad"); // se ejecuta este bloque
} else {
    console.log("Eres menor de edad");  // no se ejecuta este bloque
}

Ejemplo de switch:

let fruta = "manzana";
switch (fruta) {
    case "naranja": // resultado de la condicional es false
        console.log("La fruta es una naranja"); // no se ejecuta este bloque
        break;
    case "manzana": // resultado de la condicional es true
        console.log("La fruta es una manzana"); // se ejecuta este bloque
        break;
    default:
        console.log("La fruta no es ni naranja ni manzana");
}

Bucles: while, do while, for...of, for...in y foreach.

¿Que es un bucle?

Un bucle es una estructura de control que permite repetir una porción del código, hasta cumplir ciertas condiciones.

Tipos de bucles en JavaScript

  • Bucle while: Repite un bloque de código mientras una condición sea verdadera.
  • Bucle do...while: Similar al bucle while, pero garantiza que el bloque de código se ejecute al menos una vez antes de evaluar la condición.
  • Bucle for: Utilizado para repetir un bloque de código un número específico de veces, con una inicialización, condición y actualización en una sola línea.
  • Bucle for...of: Permite iterar sobre los valores de una colección iterable, como arrays o strings.
  • Bucle for...in: Permite iterar sobre las propiedades enumerables de un objeto.
  • Bucle foreach: Permite iterar sobre los elementos de un array, ejecutando una función para cada elemento.

Ejemplos de bucles

Ejemplo de bucle while:

let i = 0;
while (i < 5) { //mientras se cumpla la condicion se repetira el bucle.
    console.log(i);
    i++;
}

Ejemplo de bucle do...while:

let j = 0;
do {
    console.log(j);
    j++;
} while (j < 5); //se ejecuta al menos una vez y luego verifica la condición

Ejemplo de bucle for:

/* k = 0 Inicialización, 
k < 5 condicion, k++ incremento, 
cuando k = 6 deja de repetirce */
for (let k = 0; k < 5; k++) { 
    console.log(k);
}

Ejemplo de bucle for...of:

let frutas = ["manzana", "banana", "cereza"];
for (let fruta of frutas) { //itera sobre los valores del array
    console.log(fruta); //resultado: manzana banana cereza
}

Ejemplo de bucle for...if:


let persona = {nombre: "Juan", edad: 30, ciudad: "Madrid"};
// nombre => clave, "Juan" => valor
for (let clave in persona) { //itera sobre las propiedades del objeto
    console.log(clave + ": " + persona[clave]); // Resultado: nombre: Juan, edad: 30, ciudad: Madrid
}

Ejemplo de bucle foreach:


let numeros = [1, 2, 3, 4, 5];
numeros.forEach(function(numero) { //ejecuta la función para cada elemento del array
    console.log(numero + 1); // Resultado: 2, 3, 4, 5, 6
});

Control de Flujo: break, continue

¿Que es el control de flujo?

El control de flujo se refiere a las estructuras y mecanismos que permiten alterar la secuencia normal de ejecución del código en un programa. En JavaScript, dos de los principales mecanismos de control de flujo son las declaraciones break y continue.

Declaración break:

La declaración break se utiliza para salir inmediatamente de un bucle o una estructura switch. Cuando se encuentra una declaración break dentro de un bucle, la ejecución del bucle se detiene y el control se transfiere a la siguiente línea de código después del bucle.

Ejemplo de declaración break:


let i = 0;
while (i < 10) {
    if (i === 5) {
         break; // Sale del bucle cuando i es igual a 5
    }
    console.log(i);
    i++;
} // Resultado: 0, 1, 2, 3, 4

Declaración continue:

La declaración continue se utiliza para saltar la iteración actual de un bucle y pasar a la siguiente iteración. Cuando se encuentra una declaración continue dentro de un bucle, el resto del código dentro del bucle para esa iteración se omite y el control vuelve al inicio del bucle para la siguiente iteración.

Ejemplo de declaración continue:


for (let j = 0; j < 10; j++) {
    if (j % 2 === 0) {
      continue; // Salta a la siguiente iteración si j es par
    }
    console.log(j); // Solo imprime números impares Resultado: 1, 3, 5, 7, 9
}
