Objetos y Arreglos.

En este módulo, exploraremos los conceptos básicos de JavaScript, incluyendo objetos, arreglos y sus métodos.

Temas del Módulo

Creación y manipulación de Objetos.

Los objetos son colecciones de propiedades y métodos. Se pueden crear utilizando llaves {} y se pueden manipular agregando, modificando o eliminando propiedades.

Propiedades y métodos de Objetos.

Las propiedades son valores asociados a un objeto, mientras que los métodos son funciones asociadas a un objeto. Se accede a las propiedades y métodos utilizando la notación de punto o corchetes.

Métodos comunes de Objetos.

JavaScript proporciona varios métodos integrados para trabajar con objetos, como Object.keys(), Object.values() y Object.entries().

Creación y manipulación de Arreglos.

Los arreglos son listas ordenadas de valores. Se pueden crear utilizando corchetes [] y se pueden manipular agregando, eliminando o modificando elementos.

Métodos de arrays: map, filter, reduce, find, forEach, etc.

JavaScript proporciona varios métodos integrados para trabajar con arreglos, como Array.prototype.map(), Array.prototype.filter() y Array.prototype.reduce().

Desestructuración y operador spread/rest.

La desestructuración permite extraer valores de objetos o arreglos en variables individuales. El operador spread/rest (...) se utiliza para expandir o agrupar elementos.

Iteración con for...of y for...in.

El bucle for...of se utiliza para iterar sobre los valores de un arreglo, mientras que el bucle for...in se utiliza para iterar sobre las propiedades enumerables de un objeto.

Creación y manipulación de Objetos

¿Qué es un Objeto en JavaScript?

Un objeto es una colección de propiedades relacionadas. Cada propiedad es una asociación entre un nombre (clave) y un valor. Los objetos son uno de los tipos de datos más importantes en JavaScript y permiten agrupar información relacionada.

💡 Analogía: Piensa en un objeto como una ficha de información de una persona. La ficha tiene campos (nombre, edad, email) y cada campo tiene su valor específico.

Formas de Crear Objetos

1. Notación de Objeto Literal (la más común)

Sintaxis con llaves {} 
// Objeto vacío
let objetoVacio = {};

// Objeto con propiedades
let persona = {
    nombre: 'Juan',
    edad: 30,
    ciudad: 'Madrid',
    esEstudiante: false
};

console.log(persona);
// { nombre: 'Juan', edad: 30, ciudad: 'Madrid', esEstudiante: false }

2. Usando el Constructor Object()

Constructor Object 
// Menos común, pero válido
let persona = new Object();
persona.nombre = 'María';
persona.edad = 25;
persona.ciudad = 'Barcelona';

console.log(persona);
// { nombre: 'María', edad: 25, ciudad: 'Barcelona' }

3. Usando Object.create()

Crear con prototipo 
// Crear un objeto basado en otro
let personaBase = {
    tipo: 'Humano',
    saludar: function() {
        console.log('¡Hola!');
    }
};

let estudiante = Object.create(personaBase);
estudiante.nombre = 'Carlos';
estudiante.carrera = 'Ingeniería';

console.log(estudiante.tipo); // 'Humano' (heredado)
estudiante.saludar(); // ¡Hola! (heredado)

Propiedades de los Objetos

Propiedades de un Objeto

Las propiedades pueden ser de cualquier tipo de dato:

Diferentes tipos de valores 
let usuario = {
    // String
    nombre: 'Ana',
    
    // Number
    edad: 28,
    puntos: 1500,
    
    // Boolean
    activo: true,
    premium: false,
    
    // Array
    hobbies: ['leer', 'correr', 'programar'],
    
    // Objeto anidado
    direccion: {
        calle: 'Av. Principal',
        numero: 123,
        ciudad: 'Valencia'
    },
    
    // Función (método)
    saludar: function() {
        console.log('Hola, soy ' + this.nombre);
    },
    
    // null
    foto: null,
    
    // undefined
    telefono: undefined
};

console.log(usuario);

Acceder a las Propiedades

1. Notación de Punto (.)

Sintaxis: objeto.propiedad

let coche = {
    marca: 'Toyota',
    modelo: 'Corolla',
    año: 2023
};

// Leer propiedades
console.log(coche.marca);   // 'Toyota'
console.log(coche.modelo);  // 'Corolla'
console.log(coche.año);     // 2023

2. Notación de Corchetes []

Sintaxis: objeto['propiedad'] 
let coche = {
    marca: 'Toyota',
    modelo: 'Corolla',
    'año de fabricación': 2023  // Propiedad con espacios
};

// Leer con corchetes
console.log(coche['marca']);    // 'Toyota'
console.log(coche['modelo']);   // 'Corolla'

// Necesario para propiedades con espacios o caracteres especiales
console.log(coche['año de fabricación']); // 2023

// Útil con variables
let propiedad = 'marca';
console.log(coche[propiedad]); // 'Toyota'

⚠️ Diferencias importantes:

  • Punto (.): Más simple y legible, pero no funciona con espacios o variables
  • Corchetes []: Más flexible, funciona con strings dinámicos y nombres con espacios

Agregar y Modificar Propiedades

Añadir nuevas propiedades

let libro = {
    titulo: '1984',
    autor: 'George Orwell'
};

// Agregar nueva propiedad con punto
libro.año = 1949;
libro.idioma = 'Inglés';

// Agregar con corchetes
libro['páginas'] = 328;
libro['editorial'] = 'Penguin Books';

console.log(libro);
/*
{
  titulo: '1984',
  autor: 'George Orwell',
  año: 1949,
  idioma: 'Inglés',
  páginas: 328,
  editorial: 'Penguin Books'
}
*/

Modificar propiedades existentes

let producto = {
    nombre: 'Laptop',
    precio: 800,
    stock: 10
};

console.log(producto.precio); // 800

// Modificar precio
producto.precio = 750;
console.log(producto.precio); // 750

// Modificar stock
producto.stock = producto.stock - 1; // 9
// o
producto.stock--; // 8

console.log(producto);
// { nombre: 'Laptop', precio: 750, stock: 8 }

Eliminar Propiedades

Operador delete

let usuario = {
    nombre: 'Pedro',
    email: 'pedro@email.com',
    contraseña: '12345',
    edad: 30
};

console.log(usuario);
// { nombre: 'Pedro', email: 'pedro@email.com', contraseña: '12345', edad: 30 }

// Eliminar propiedad sensible
delete usuario.contraseña;

console.log(usuario);
// { nombre: 'Pedro', email: 'pedro@email.com', edad: 30 }

// Verificar si existe
console.log(usuario.contraseña); // undefined
💡 Nota: Usar delete elimina completamente la propiedad. Asignar undefined solo cambia el valor pero la propiedad sigue existiendo.

Verificar si una Propiedad Existe

Diferentes métodos de verificación

let persona = {
    nombre: 'Laura',
    edad: 25,
    ciudad: 'Sevilla'
};

// 1. Operador 'in'
console.log('nombre' in persona);    // true
console.log('apellido' in persona);  // false

// 2. hasOwnProperty()
console.log(persona.hasOwnProperty('edad'));     // true
console.log(persona.hasOwnProperty('país'));     // false

// 3. Verificar valor !== undefined
console.log(persona.ciudad !== undefined);   // true
console.log(persona.teléfono !== undefined); // false

// 4. Usando el operador ?.  (Optional Chaining)
console.log(persona?.nombre);     // 'Laura'
console.log(persona?.apellido);   // undefined

Métodos de Objetos

Métodos en Objetos

Un método es una función que es propiedad de un objeto:

Definir métodos

let calculadora = {
    // Propiedades
    marca: 'Casio',
    modelo: 'FX-82',
    
    // Métodos
    sumar: function(a, b) {
        return a + b;
    },
    
    restar: function(a, b) {
        return a - b;
    },
    
    // Sintaxis moderna (ES6)
    multiplicar(a, b) {
        return a * b;
    },
    
    dividir(a, b) {
        if (b === 0) {
            return 'Error: No se puede dividir por cero';
        }
        return a / b;
    }
};

// Llamar métodos
console.log(calculadora.sumar(5, 3));        // 8
console.log(calculadora.restar(10, 4));      // 6
console.log(calculadora.multiplicar(6, 7));  // 42
console.log(calculadora.dividir(20, 4));     // 5
console.log(calculadora.dividir(10, 0));     // 'Error: No se puede dividir por cero'

Usando 'this' en métodos

let cuenta = {
    titular: 'María García',
    saldo: 1000,
    moneda: 'EUR',
    
    // Método para consultar saldo
    consultarSaldo() {
        return `Saldo de ${this.titular}: ${this.saldo} ${this.moneda}`;
    },
    
    // Método para depositar
    depositar(cantidad) {
        if (cantidad > 0) {
            this.saldo += cantidad;
            return `Depósito exitoso. Nuevo saldo: ${this.saldo} ${this.moneda}`;
        }
        return 'Cantidad inválida';
    },
    
    // Método para retirar
    retirar(cantidad) {
        if (cantidad > this.saldo) {
            return 'Fondos insuficientes';
        }
        if (cantidad > 0) {
            this.saldo -= cantidad;
            return `Retiro exitoso. Nuevo saldo: ${this.saldo} ${this.moneda}`;
        }
        return 'Cantidad inválida';
    }
};

console.log(cuenta.consultarSaldo());
// 'Saldo de María García: 1000 EUR'

console.log(cuenta.depositar(500));
// 'Depósito exitoso. Nuevo saldo: 1500 EUR'

console.log(cuenta.retirar(300));
// 'Retiro exitoso. Nuevo saldo: 1200 EUR'

console.log(cuenta.consultarSaldo());
// 'Saldo de María García: 1200 EUR'
✅ this en Objetos: La palabra clave this se refiere al objeto actual. Dentro de un método, this apunta al objeto que contiene ese método.

Objetos Anidados

Los objetos pueden contener otros objetos:

Estructura jerárquica

let empresa = {
    nombre: 'TechCorp',
    fundacion: 2020,
    empleados: 150,
    
    // Objeto anidado: dirección
    direccion: {
        calle: 'Av. Tecnología',
        numero: 456,
        ciudad: 'Madrid',
        codigoPostal: '28001',
        pais: 'España'
    },
    
    // Objeto anidado: CEO
    ceo: {
        nombre: 'Carlos Pérez',
        edad: 45,
        email: 'carlos@techcorp.com',
        
        // Objeto dentro de objeto dentro de objeto
        contacto: {
            telefono: '+34 600 123 456',
            linkedin: 'carlos-perez'
        }
    },
    
    // Array de objetos
    departamentos: [
        { nombre: 'Ventas', personal: 30 },
        { nombre: 'IT', personal: 50 },
        { nombre: 'Marketing', personal: 20 }
    ]
};

// Acceder a propiedades anidadas
console.log(empresa.nombre);                    // 'TechCorp'
console.log(empresa.direccion.ciudad);          // 'Madrid'
console.log(empresa.ceo.nombre);                // 'Carlos Pérez'
console.log(empresa.ceo.contacto.telefono);     // '+34 600 123 456'
console.log(empresa.departamentos[0].nombre);   // 'Ventas'
console.log(empresa.departamentos[1].personal); // 50

Optional Chaining (?.) para acceso seguro

let usuario = {
    nombre: 'Ana',
    configuracion: {
        notificaciones: true
    }
};

// Sin optional chaining (puede causar error)
// console.log(usuario.perfil.foto); // ❌ Error: Cannot read property 'foto' of undefined

// Con optional chaining (seguro)
console.log(usuario.perfil?.foto);              // undefined (no error)
console.log(usuario.configuracion?.tema);       // undefined (no error)
console.log(usuario.configuracion?.notificaciones); // true

// Encadenar múltiples niveles
console.log(usuario.direccion?.ciudad?.nombre); // undefined (no error)

Copiar Objetos

⚠️ Importante: Los objetos se pasan por referencia, no por valor. Esto significa que al asignar un objeto a otra variable, ambas apuntan al mismo objeto en memoria.

Copia por referencia (NO es una copia real)

let original = { nombre: 'Juan', edad: 30 };
let copia = original;  // ❌ No es una copia, es la misma referencia

copia.edad = 31;

console.log(original.edad); // 31 (¡se modificó el original!)
console.log(copia.edad);    // 31

Métodos para Copiar Objetos

1. Spread Operator (...) - Copia superficial

let original = { nombre: 'María', edad: 25, ciudad: 'Madrid' };
let copia = { ...original };  // ✅ Copia real (superficial)

copia.edad = 26;

console.log(original.edad); // 25 (original sin cambios)
console.log(copia.edad);    // 26

// LIMITACIÓN: Solo copia el primer nivel
let persona = {
    nombre: 'Carlos',
    direccion: { ciudad: 'Barcelona' }
};

let copiaPersona = { ...persona };
copiaPersona.direccion.ciudad = 'Valencia';

console.log(persona.direccion.ciudad); // 'Valencia' (¡se modificó!)
// Los objetos anidados aún comparten referencia

2. Object.assign() - Copia superficial

let original = { a: 1, b: 2 };
let copia = Object.assign({}, original);

copia.b = 3;

console.log(original.b); // 2
console.log(copia.b);    // 3

3. JSON.parse + JSON.stringify - Copia profunda

let original = {
    nombre: 'Ana',
    edad: 28,
    direccion: {
        ciudad: 'Sevilla',
        pais: 'España'
    },
    hobbies: ['leer', 'nadar']
};

// Copia profunda (todos los niveles)
let copiaCompleta = JSON.parse(JSON.stringify(original));

copiaCompleta.direccion.ciudad = 'Granada';
copiaCompleta.hobbies.push('correr');

console.log(original.direccion.ciudad);  // 'Sevilla' (sin cambios)
console.log(copiaCompleta.direccion.ciudad); // 'Granada'

console.log(original.hobbies);       // ['leer', 'nadar'] (sin cambios)
console.log(copiaCompleta.hobbies);  // ['leer', 'nadar', 'correr']

// LIMITACIÓN: No funciona con funciones, undefined, Date, etc.

Comparar Objetos

Comparación de objetos

let obj1 = { nombre: 'Juan', edad: 30 };
let obj2 = { nombre: 'Juan', edad: 30 };
let obj3 = obj1;  // misma referencia

// Comparación por referencia
console.log(obj1 === obj2);  // false (diferentes objetos en memoria)
console.log(obj1 === obj3);  // true (misma referencia)

// Comparar propiedades manualmente
function objetosIguales(a, b) {
    return a.nombre === b.nombre && a.edad === b.edad;
}

console.log(objetosIguales(obj1, obj2)); // true

// Comparar con JSON (solo para objetos simples)
console.log(JSON.stringify(obj1) === JSON.stringify(obj2)); // true

Creación de Arreglos

Los arreglos se pueden crear de varias formas:

1. Notación de Corchetes []

// Arreglo vacío
    let arregloVacio = [];
    // Arreglo con elementos
    let arregloConElementos = [1, 2, 3, 4, 5];
    console.log(arregloConElementos); // [1, 2, 3, 4, 5]

2. Usando el Constructor Array()

// Arreglo vacío
    let arregloVacio = new Array();
    // Arreglo con elementos
    let arregloConElementos = new Array(1, 2, 3, 4, 5);
    console.log(arregloConElementos); // [1, 2, 3, 4, 5]

3. Usando Array.of()

// Crear arreglo con elementos
    let arregloConElementos = Array.of(1, 2, 3, 4, 5);
    console.log(arregloConElementos); // [1, 2, 3, 4, 5]

4. Usando Array.from()

// Crear arreglo a partir de un objeto iterable
    let arregloDesdeString = Array.from('Hola');
    console.log(arregloDesdeString); // ['H', 'o', 'l', 'a']

✅ Resumen:

  • Notación de corchetes []: Forma más común y sencilla.
  • Constructor Array(): Menos común, pero válido.
  • Array.of(): Útil para crear arreglos con un número variable de elementos.
  • Array.from(): Convierte objetos iterables en arreglos.

Métodos de Arreglos

Metodo: map

El método map() crea un nuevo arreglo con los resultados de la llamada a la función proporcionada en cada elemento del arreglo.

// Ejemplo de uso
let numeros = [1, 2, 3, 4, 5];
let cuadrados = numeros.map(function(num) {
    return num * num;
});
console.log(cuadrados); // [1, 4, 9, 16, 25]

En este ejemplo, map() se utiliza para crear un nuevo arreglo cuadrados que contiene los cuadrados de los números en el arreglo original numeros.

Metodo: filter

El método filter() crea un nuevo arreglo con todos los elementos que cumplan la condición implementada por la función proporcionada.

// Ejemplo de uso
let numeros = [1, 2, 3, 4, 5];
let pares = numeros.filter(function(num) {
    return num % 2 === 0;
});
console.log(pares); // [2, 4]

En este ejemplo, filter() se utiliza para crear un nuevo arreglo pares que contiene solo los números pares del arreglo original numeros.

Metodo: reduce

El método reduce() ejecuta una función reductora sobre cada elemento de un arreglo, devolviendo como resultado un único valor.

// Ejemplo de uso
let numeros = [1, 2, 3, 4, 5];
let suma = numeros.reduce(function(acumulador, num) {
    return acumulador + num;
}, 0);
console.log(suma); // 15

En este ejemplo, reduce() se utiliza para calcular la suma de todos los números en el arreglo numeros.

Metodo: find

El método find() devuelve el primer elemento de un arreglo que cumple con la condición proporcionada en la función de prueba.

// Ejemplo de uso
let numeros = [1, 2, 3, 4, 5];
let primerPar = numeros.find(function(num) {
    return num % 2 === 0;
});
console.log(primerPar); // 2

En este ejemplo, find() se utiliza para encontrar el primer número par en el arreglo numeros.

Metodo: forEach

El método forEach() ejecuta una función proporcionada una vez por cada elemento del arreglo.

// Ejemplo de uso
let numeros = [1, 2, 3, 4, 5];
numeros.forEach(function(num) {
    console.log(num);
});

En este ejemplo, forEach() se utiliza para imprimir cada número en el arreglo numeros.

Destructuración y Operadores Rest

Destructuración de Objetos

La destructuración permite extraer propiedades de un objeto y asignarlas a variables individuales.

// Ejemplo de uso
let persona = {
    nombre: 'Juan',
    edad: 30,
    ciudad: 'Madrid'
};
let { nombre, edad } = persona;
console.log(nombre); // Juan
console.log(edad);   // 30

Operadores Rest

El operador rest (...) permite agrupar el resto de las propiedades en un nuevo objeto.

// Ejemplo de uso
let persona = {
    nombre: 'Juan',
    edad: 30,
    ciudad: 'Madrid'
};
let { nombre, ...rest } = persona;
console.log(nombre); // Juan
console.log(resto);   // { edad: 30, ciudad: 'Madrid' }

Operadores Spread

El operador spread (...) permite expandir las propiedades de un objeto en otro objeto.

// Ejemplo de uso
let persona = {
    nombre: 'Juan',
    edad: 30,
    ciudad: 'Madrid'
};
let nuevaPersona = { ...persona, pais: 'España' };
console.log(nuevaPersona); // { nombre: 'Juan', edad: 30, ciudad: 'Madrid', pais: 'España' }

Iteración con for...of y for...in

Bucle for...of

El bucle for...of se utiliza para iterar sobre los valores de un objeto iterable, como un array o una cadena de texto.

// Ejemplo con un array
let numeros = [1, 2, 3, 4, 5];
for (let numero of numeros) {
    console.log(numero);
}

Bucle for...in

El bucle for...in se utiliza para iterar sobre las propiedades enumerables de un objeto.

// Ejemplo con un objeto
let persona = {
    nombre: 'Juan',
    edad: 30,
    ciudad: 'Madrid'
};
for (let propiedad in persona) {
    console.log(propiedad + ': ' + persona[propiedad]);
}

Ejercicio Práctico: Sistema de Biblioteca

Intermedio

Objetivo:

Crear un sistema de gestión de libros usando objetos.

Requisitos:

  1. Crea un objeto biblioteca con:
    • nombre de la biblioteca
    • dirección
    • array de libros (cada libro es un objeto)
    • métodos para agregar, buscar y listar libros
  2. Cada libro debe tener: título, autor, año, disponible (boolean)
  3. Implementa los siguientes métodos:
    • agregarLibro(libro)
    • buscarPorAutor(autor)
    • listarDisponibles()
    • prestarLibro(titulo)

Solución propuesta

let biblioteca = {
    nombre: 'Biblioteca Municipal',
    direccion: 'Calle Principal 123',
    libros: [
        {
            titulo: '1984',
            autor: 'George Orwell',
            año: 1949,
            disponible: true
        },
        {
            titulo: 'Cien años de soledad',
            autor: 'Gabriel García Márquez',
            año: 1967,
            disponible: true
        }
    ],
    
    agregarLibro(libro) {
        this.libros.push(libro);
        console.log(`Libro "${libro.titulo}" agregado correctamente`);
    },
    
    buscarPorAutor(autor) {
        return this.libros.filter(libro => 
            libro.autor.toLowerCase().includes(autor.toLowerCase())
        );
    },
    
    listarDisponibles() {
        console.log('Libros disponibles:');
        this.libros
            .filter(libro => libro.disponible)
            .forEach(libro => {
                console.log(`- ${libro.titulo} (${libro.autor})`);
            });
    },
    
    prestarLibro(titulo) {
        let libro = this.libros.find(l => l.titulo === titulo);
        
        if (!libro) {
            return 'Libro no encontrado';
        }
        
        if (!libro.disponible) {
            return 'El libro ya está prestado';
        }
        
        libro.disponible = false;
        return `Libro "${titulo}" prestado exitosamente`;
    }
};

// Probar el sistema
biblioteca.listarDisponibles();

console.log(biblioteca.prestarLibro('1984'));
// 'Libro "1984" prestado exitosamente'

biblioteca.listarDisponibles();
// Ya no muestra '1984'

let librosGarcia = biblioteca.buscarPorAutor('García');
console.log(librosGarcia);
// [{ titulo: 'Cien años de soledad', ... }]
