04

Objetos y Arreglos.

En este módulo, exploraremos los conceptos básicos de JavaScript, incluyendo objetos, arreglos y sus métodos.

Temas del Módulo

Creación y manipulación de Objetos.

Los objetos son colecciones de propiedades y métodos. Se pueden crear utilizando llaves {} y se pueden manipular agregando, modificando o eliminando propiedades.

Propiedades y métodos de Objetos.

Las propiedades son valores asociados a un objeto, mientras que los métodos son funciones asociadas a un objeto. Se accede a las propiedades y métodos utilizando la notación de punto o corchetes.

Métodos comunes de Objetos.

JavaScript proporciona varios métodos integrados para trabajar con objetos, como Object.keys(), Object.values() y Object.entries().

Creación y manipulación de Arreglos.

Los arreglos son listas ordenadas de valores. Se pueden crear utilizando corchetes [] y se pueden manipular agregando, eliminando o modificando elementos.

Métodos de arrays: map, filter, reduce, find, forEach, etc.

JavaScript proporciona varios métodos integrados para trabajar con arreglos, como Array.prototype.map(), Array.prototype.filter() y Array.prototype.reduce().

Desestructuración y operador spread/rest.

La desestructuración permite extraer valores de objetos o arreglos en variables individuales. El operador spread/rest (...) se utiliza para expandir o agrupar elementos.

Iteración con for...of y for...in.

El bucle for...of se utiliza para iterar sobre los valores de un arreglo, mientras que el bucle for...in se utiliza para iterar sobre las propiedades enumerables de un objeto.
