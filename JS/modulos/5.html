Manipulacion del DOM

En este módulo, exploraremos los conceptos básicos de JavaScript, incluyendo objetos, arreglos y sus métodos.

Temas del Módulo:

¿Que es el DOM?.

El DOM (Document Object Model) es una interfaz de programación que permite a los scripts acceder y manipular el contenido, la estructura y el estilo de un documento HTML.

Selección de elementos.

Para seleccionar elementos del DOM, se pueden utilizar métodos como getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() y querySelectorAll().

Modificación de contenido y atributos.

Para modificar el contenido y los atributos de los elementos del DOM, se pueden utilizar propiedades como innerHTML, textContent y setAttribute().

Creación y eliminación de elementos.

Para crear y eliminar elementos del DOM, se pueden utilizar métodos como document.createElement() y element.remove().

Manejo de clases y estilos

Para manejar clases y estilos de los elementos del DOM, se pueden utilizar propiedades como classList y style.

Eventos.

Los eventos son acciones o sucesos que ocurren en el navegador, como clics de ratón, pulsaciones de teclas o movimientos del ratón. Se pueden manejar utilizando métodos como addEventListener().

Propagación y delegación de eventos.

La propagación de eventos se refiere al orden en que se activan los controladores de eventos en el DOM. La delegación de eventos es una técnica que consiste en agregar un solo controlador de eventos a un elemento padre en lugar de a múltiples elementos hijos.

¿Qué es el DOM?

El DOM (Document Object Model) es una interfaz de programación que permite a los scripts acceder y manipular el contenido, la estructura y el estilo de un documento HTML. El DOM representa el documento como una estructura de árbol, donde cada nodo es un objeto que representa una parte del documento, como un elemento, un atributo o un texto.

Con el DOM, los desarrolladores pueden crear, eliminar o modificar elementos HTML, cambiar estilos CSS, manejar eventos y realizar otras operaciones dinámicas en una página web.

El DOM es fundamental para el desarrollo web, ya que permite a los desarrolladores interactuar con el contenido de la página de manera dinámica y crear experiencias de usuario más ricas e interactivas.

El DOM se puede manipular utilizando JavaScript, lo que permite a los desarrolladores crear aplicaciones web interactivas y dinámicas.

Además, el DOM proporciona una forma de estructurar y organizar el contenido de una página web, lo que facilita su mantenimiento y actualización.

Ejemplo de Estructura del DOM y uso

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de DOM</title>
  </head>
    <body>
        <h1 id="titulo">Hola, Mundo!</h1>
        <script>
            // Acceder al elemento h1 por su ID y cambiar su contenido
            document.getElementById("titulo").textContent = "¡Hola, DOM!";
        </script>
    </body>
</html>

Este es un ejemplo de cómo se puede representar la estructura de un documento HTML en el DOM.

¿Cómo seleccionar elementos en el DOM?

Para seleccionar elementos en el DOM, se pueden utilizar varios métodos proporcionados por la API del DOM. Algunos de los métodos más comunes son:

  • document.getElementById(id): Selecciona un elemento por su ID.
  • document.getElementsByClassName(className): Selecciona todos los elementos que tienen una clase específica.
  • document.getElementsByTagName(tagName): Selecciona todos los elementos con un nombre de etiqueta específico.
  • document.querySelector(selector): Selecciona el primer elemento que coincide con un selector CSS.
  • document.querySelectorAll(selector): Selecciona todos los elementos que coinciden con un selector CSS.

Seleccionar por ID

El método getElementById() se utiliza para seleccionar un elemento del DOM utilizando su atributo id. Este método devuelve un solo elemento, ya que los IDs deben ser únicos en un documento HTML.

Ejemplo de getElementById()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de DOM</title>
  </head>
    <body>
        <h1 id="titulo">Hola, Mundo!</h1>
        <script>
            // Acceder al elemento h1 por su ID y cambiar su contenido
            document.getElementById("titulo").textContent = "¡Hola, DOM!";
        </script>
    </body>
</html>

Seleccionar por Clase

El método getElementsByClassName() se utiliza para seleccionar todos los elementos del DOM que tienen una clase específica. Este método devuelve una colección de elementos (HTMLCollection) que coinciden con la clase proporcionada.

Ejemplo de getElementsByClassName()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de DOM</title>
  </head>
    <body>
        <h1 class="titulo">Hola, Mundo!</h1>
        <h1 class="titulo">Hola, DOM!</h1>
        <script>
            // Acceder a todos los elementos h1 por su clase y cambiar su contenido
            var titulos = document.getElementsByClassName("titulo");
            for (var i = 0; i < titulos.length; i++) {
                titulos[i].textContent = "¡Hola, Clase!";
            }
        </script>
    </body>
</html>

Seleccionar por Nombre de Etiqueta

El método getElementsByTagName() se utiliza para seleccionar todos los elementos del DOM con un nombre de etiqueta específico. Este método devuelve una colección de elementos (HTMLCollection) que coinciden con el nombre de etiqueta proporcionado.

Ejemplo de getElementsByTagName()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de DOM</title>
  </head>
    <body>
        <h1>Hola, Mundo!</h1>
        <h1>Hola, DOM!</h1>
        <script>
            // Acceder a todos los elementos h1 por su nombre de etiqueta y cambiar su contenido
            var titulos = document.getElementsByTagName("h1");
            for (var i = 0; i < titulos.length; i++) {
                titulos[i].textContent = "¡Hola, Etiqueta!";
            }
        </script>
    </body>
</html>

Seleccionar con Query Selector

El método querySelector() se utiliza para seleccionar el primer elemento del DOM que coincide con un selector CSS proporcionado. Este método es muy flexible, ya que permite utilizar cualquier selector CSS válido.

Ejemplo de querySelector()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de DOM</title>
  </head>
    <body>
        <h1 class="titulo">Hola, Mundo!</h1>
        <h1 class="titulo">Hola, DOM!</h1>
        <script>
            // Acceder al primer elemento h1 por su clase y cambiar su contenido
            var titulo = document.querySelector(".titulo");
            titulo.textContent = "¡Hola, Query Selector!";
        </script>
    </body>
</html>

Seleccionar con Query Selector All

El método querySelectorAll() se utiliza para seleccionar todos los elementos del DOM que coinciden con un selector CSS proporcionado. Este método devuelve una NodeList de elementos que coinciden con el selector.

Ejemplo de querySelectorAll()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de DOM</title>
  </head>
    <body>
        <h1 class="titulo">Hola, Mundo!</h1>
        <h1 class="titulo">Hola, DOM!</h1>
        <script>
            // Acceder a todos los elementos h1 por su clase y cambiar su contenido
            var titulos = document.querySelectorAll(".titulo");
            for (var i = 0; i < titulos.length; i++) {
                titulos[i].textContent = "¡Hola, Query Selector All!";
            }
        </script>
    </body>
</html>

Modificación de contenido y atributos

Para modificar el contenido y los atributos de los elementos del DOM, se pueden utilizar varias propiedades y métodos proporcionados por la API del DOM. Algunos de los más comunes son:

  • innerHTML: Permite obtener o establecer el contenido HTML de un elemento.
  • textContent: Permite obtener o establecer el contenido de texto de un elemento.
  • setAttribute(): Permite establecer el valor de un atributo específico en un elemento.
  • getAttribute(): Permite obtener el valor de un atributo específico en un elemento.
  • removeAttribute(): Permite eliminar un atributo específico de un elemento.

innerHTML

La propiedad innerHTML permite obtener o establecer el contenido HTML de un elemento. Al establecer esta propiedad, se reemplaza todo el contenido existente del elemento con el nuevo contenido proporcionado.

Ejemplo de innerHTML

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de innerHTML</title>
  </head>
    <body>
        <div id="contenido"></div>
        <script>
            // Establecer el contenido HTML del div
            document.getElementById("contenido").innerHTML = "<h1>Hola, Mundo!</h1><p>Este es un párrafo.</p>";
        </script>
    </body>
</html>

textContent

La propiedad textContent permite obtener o establecer el contenido de texto de un elemento. A diferencia de innerHTML, esta propiedad no interpreta el contenido como HTML, sino que lo trata como texto plano.

Ejemplo de textContent

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de textContent</title>
  </head>
    <body>
        <div id="contenido"></div>
        <script>
            // Establecer el contenido de texto del div
            document.getElementById("contenido").textContent = "Hola, Mundo! Este es un párrafo.";
        </script>
    </body>
</html>

setAttribute

El método setAttribute() permite establecer el valor de un atributo específico en un elemento del DOM. Si el atributo ya existe, su valor se actualizará; si no existe, se creará.

Ejemplo de setAttribute()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de setAttribute()</title>
  </head>
  <body>
    <div id="contenido"></div>
    <script>
      // Establecer un atributo en el div
      document.getElementById("contenido").setAttribute("data-info", "Este es un div.");
    </script>
  </body>
</html>

getAttribute

El método getAttribute() permite obtener el valor de un atributo específico en un elemento del DOM.

Ejemplo de getAttribute()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de getAttribute()</title>
  </head>
  <body>
    <div id="contenido" data-info="Este es un div."></div>
    <script>
      // Obtener el valor de un atributo en el div
      var info = document.getElementById("contenido").getAttribute("data-info");
      console.log(info);
    </script>
  </body>
</html>

removeAttribute

El método removeAttribute() permite eliminar un atributo específico de un elemento del DOM.

Ejemplo de removeAttribute()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de removeAttribute()</title>
  </head>
  <body>
    <div id="contenido" data-info="Este es un div."></div>

Creación y eliminación de elementos

Para crear y eliminar elementos del DOM, se pueden utilizar varios métodos proporcionados por la API del DOM. Algunos de los más comunes son:

  • document.createElement(tagName): Crea un nuevo elemento con el nombre de etiqueta especificado.
  • element.appendChild(childElement): Agrega un elemento hijo al final de la lista de hijos de un elemento padre.
  • element.removeChild(childElement): Elimina un elemento hijo específico de un elemento padre.
  • element.remove(): Elimina el elemento del DOM.

createElement

El método createElement() se utiliza para crear un nuevo elemento del DOM con el nombre de etiqueta especificado.

Ejemplo de createElement()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de createElement()</title>
  </head>
    <body>
        <div id="contenido"></div>
        <script>
            // Crear un nuevo elemento p
            var nuevoParrafo = document.createElement("p");
            nuevoParrafo.textContent = "Este es un nuevo párrafo.";
            document.getElementById("contenido").appendChild(nuevoParrafo);
        </script>
    </body>
</html>

appendChild

El método appendChild() se utiliza para agregar un elemento hijo al final de la lista de hijos de un elemento padre.

Ejemplo de appendChild()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de appendChild()</title>
  </head>
    <body>
        <div id="contenido"></div>
        <script>
            // Crear un nuevo elemento p y agregarlo al div
            var nuevoParrafo = document.createElement("p");
            nuevoParrafo.textContent = "Este es un nuevo párrafo.";
            document.getElementById("contenido").appendChild(nuevoParrafo);
        </script>
    </body>
</html>

removeChild

El método removeChild() se utiliza para eliminar un elemento hijo específico de un elemento padre.

Ejemplo de removeChild()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de removeChild()</title>
  </head>
  <body>
    <div id="contenido">
      <p>Este es un párrafo que será eliminado.</p>
    </div>
    <script>
      // Eliminar el párrafo del div
      var parrafo = document.querySelector("#contenido p");
      document.getElementById("contenido").removeChild(parrafo);

remove

El método remove() se utiliza para eliminar un elemento del DOM.

Manejo de clases y estilos

Para manejar clases y estilos de los elementos del DOM, se pueden utilizar varias propiedades y métodos proporcionados por la API del DOM. Algunos de los más comunes son:

  • classList: Permite agregar, eliminar o alternar clases en un elemento.
  • style: Permite acceder y modificar los estilos CSS en línea de un elemento.

classList

La propiedad classList proporciona métodos para agregar, eliminar o alternar clases en un elemento del DOM.

Ejemplo de classList

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de classList</title>
  </head>
  <body>
    <div id="contenido">
      <p id="parrafo">Este es un párrafo con una clase.</p>
    </div>

      parrafo.classList.add("nueva-clase");
    </script>

style

La propiedad style permite acceder y modificar los estilos CSS en línea de un elemento del DOM.

Ejemplo de style

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de style</title>
  </head>
  <body>
    <div id="contenido">
      <p id="parrafo">Este es un párrafo con un estilo en línea.</p>

      var parrafo = document.getElementById("parrafo");
      parrafo.style.color = "red";

    </script>
  </body>
</html>

Eventos

Los eventos son acciones o sucesos que ocurren en el navegador, como clics de ratón, pulsaciones de teclas o movimientos del ratón. En el DOM, los eventos se pueden manejar utilizando métodos como addEventListener(), que permite agregar controladores de eventos a los elementos del DOM.

Ejemplo de manejo de eventos con addEventListener()

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de eventos</title>
  </head>
  <body>
    <div id="contenido">
      <p id="parrafo">Haz clic en este párrafo.</p>
    </div>
    <script>
      // Agregar un manejador de eventos al párrafo
      var parrafo = document.getElementById("parrafo");
      parrafo.addEventListener("click", function() {
        alert("¡Párrafo clickeado!");
      });
    </script>
  </body>
</html>

Eventos mousedown, mouseup, mousemove, mouseover, mouseout, mouseenter, mouseleave, contextmenu

Los eventos mousedown, mouseup, mousemove, mouseover, mouseout, mouseenter, mouseleave y contextmenu se activan en diferentes interacciones del ratón. Se pueden manejar utilizando el método addEventListener().

Ejemplo de eventos de ratón: mousedown

<!DOCTYPE html>
<html>
  <head>
    <title>Ejemplo de eventos de ratón: mousedown</title>
  </head>
  <body>
    <div id="contenido">
      <p id="parrafo">Haz clic y mantén presionado este párrafo.</p>
    </div>

      // Agregar un manejador de eventos al párrafo
      var parrafo = document.getElementById("parrafo");
      parrafo.addEventListener("mousedown", function() {
        alert("¡Ratón presionado sobre el párrafo!");
      });
    </script>
  </body>
</html>
